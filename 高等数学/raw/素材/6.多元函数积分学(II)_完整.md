## 第一节 对坐标的曲线积分的概念与性质

### 一、引例 —— 变力沿曲线做的功

**问题**：一个物体在变力 $\vec{F}(x, y)$ 作用下，沿曲线 $L$ 从 $A$ 移动到 $B$，求力做的功。

**思路**：把 $L$ 切成小段，每小段近似直线段，力近似不变 → 算出每小段的功 → 求和取极限。

设第 $i$ 小段的方向余弦为 $(\cos\alpha_i, \cos\beta_i)$，长度 $\Delta s_i$，则该小段上力做的功近似为
$$\Delta W_i \approx F_x \Delta x_i + F_y \Delta y_i = F_x \cos\alpha_i \Delta s_i + F_y \cos\beta_i \Delta s_i$$

求和取极限：
$$W = \lim_{\lambda \to 0} \sum_{i=1}^n \left[F_x(\xi_i, \eta_i) \Delta x_i + F_y(\xi_i, \eta_i) \Delta y_i\right]$$

这种"先乘坐标增量再求和"的极限，就是**对坐标的曲线积分**。

### 二、对坐标的曲线积分的概念

设 $L$ 是 $xOy$ 面上从 $A$ 到 $B$ 的有向光滑曲线弧，$P(x, y), Q(x, y)$ 在 $L$ 上有界。把 $L$ 任意分成 $n$ 段有向小弧 $\Delta L_i$（方向与 $L$ 一致），$\Delta x_i, \Delta y_i$ 是各段在两坐标轴上的投影，任取 $(\xi_i, \eta_i) \in \Delta L_i$，作积分和
$$\sum_{i=1}^n \left[P(\xi_i, \eta_i) \Delta x_i + Q(\xi_i, \eta_i) \Delta y_i\right]$$

若极限存在（与分法、取点无关），记作
$$\int_L P(x, y)\, dx + Q(x, y)\, dy = \lim_{\lambda \to 0} \sum_{i=1}^n \left[P(\xi_i, \eta_i) \Delta x_i + Q(\xi_i, \eta_i) \Delta y_i\right]$$

各部分叫法：
- $P(x, y), Q(x, y)$：**被积函数**
- $dx, dy$：**有向弧元素在坐标轴上的投影**（注意：$dx$ 可以是负的！）
- $L$：**积分弧段**（有起点和终点的有向曲线）

**常用形式**：
$$\int_L P(x, y)\, dx \quad (\text{对 } x \text{ 的曲线积分}), \quad \int_L Q(x, y)\, dy \quad (\text{对 } y \text{ 的曲线积分})$$

**空间情形**：$\int_L P\, dx + Q\, dy + R\, dz$

> 🔑 **直观理解**：被积函数乘以**坐标的微小增量**（$dx, dy, dz$）再求和。
> - 注意 $dx$ 可正可负（沿曲线走，$x$ 可能增加也可能减少），所以积分值与**方向**有关。

**第二型曲线积分** = 对坐标的曲线积分。

> 💡 **与第一型（对弧长）的对比**：
> - 第一型 $\int_L f\, ds$：$ds > 0$（弧长总为正），与方向无关
> - 第二型 $\int_L P\, dx + Q\, dy$：$dx$ 可负，与**方向**有关（$\int_{-L} = -\int_L$）

### 三、对坐标的曲线积分的性质

1. **分段可加**：若 $L = L_1 + L_2$（$L_1$ 终点 = $L_2$ 起点），则
$$\int_L P\, dx + Q\, dy = \int_{L_1} P\, dx + Q\, dy + \int_{L_2} P\, dx + Q\, dy$$

2. **方向相反变号**：若 $-L$ 是与 $L$ 方向相反的有向曲线弧，则
$$\int_{-L} P\, dx + Q\, dy = -\int_L P\, dx + Q\, dy$$

## 第二节 对坐标的曲线积分的计算

> 📌 **本节是重点**——用户要求详细讲解：什么时候对 $x$ 积分，什么时候对 $y$ 积分，每一步都要清楚。

### 核心思想

> 🔑 **对坐标的曲线积分 = 化为定积分**。

把曲线 $L$ 用参数表示，把 $dx, dy$ 都用参数表示，曲线积分就变成对参数的定积分。

### 公式（参数方程情形）

设 $L: x = \varphi(t), y = \psi(t)$，当 $t$ 从 $\alpha$ 单调变到 $\beta$ 时，曲线从 $A$ 到 $B$。若 $\varphi, \psi$ 在 $[\alpha, \beta]$ 上具有连续导数，且 $\varphi'^2(t) + \psi'^2(t) \neq 0$，则
$$\int_L P\, dx + Q\, dy = \int_\alpha^\beta \left[P(\varphi(t), \psi(t)) \varphi'(t) + Q(\varphi(t), \psi(t)) \psi'(t)\right] dt$$

> ⚠️ **关键**：积分上下限 $\alpha \to \beta$ **按曲线方向**写（不要求 $\alpha < \beta$！起点对应的参数在下，终点对应的参数在上）。

### 公式（其他情形）

**$y = f(x)$**：视作参数 $x$，起点 $x = a$，终点 $x = b$：
$$\int_L P\, dx + Q\, dy = \int_a^b \left[P(x, f(x)) + Q(x, f(x)) f'(x)\right] dx$$

**$x = g(y)$**：视作参数 $y$，起点 $y = c$，终点 $y = d$：
$$\int_L P\, dx + Q\, dy = \int_c^d \left[P(g(y), y) g'(y) + Q(g(y), y)\right] dy$$

**空间情形** $L: x = \varphi(t), y = \psi(t), z = \omega(t)$：
$$\int_L P\, dx + Q\, dy + R\, dz = \int_\alpha^\beta \left[P \varphi' + Q \psi' + R \omega'\right] dt$$

### 做题套路（逐步讲解）

> 🔑 **对坐标曲线积分四步走**：
> 1. **看曲线 $L$ 怎么给的** → 选参数化方式
> 2. **算 $dx, dy$（或 $dx, dy, dz$）**—— 用参数表示
> 3. **代被积函数**，把 $x, y, dx, dy$ 都换掉
> 4. **按曲线方向定上下限**（起点在下，终点在上）

### 例题（按步骤详细讲解）

**例 6.1**（基础，参数方程）计算 $\int_L xy\, dx$，其中 $L$ 是抛物线 $y = x^2$ 从 $A(1, 1)$ 到 $B(2, 4)$ 的一段。

**步骤 1：选参数**。
曲线由 $y = x^2$ 给出 → 视作参数 $x$。

**步骤 2：算 $dx, dy$**。
参数 $x$ → $dx = dx$，$y = x^2$ → $dy = 2x\, dx$。

**步骤 3：代被积函数**。
$xy = x \cdot x^2 = x^3$
所以 $xy\, dx = x^3\, dx$

**步骤 4：定上下限**。
从 $A(1, 1)$ 到 $B(2, 4)$ → $x$ 从 $1$ 到 $2$。

$$\int_L xy\, dx = \int_1^2 x^3\, dx = \frac{x^4}{4}\bigg|_1^2 = \frac{16}{4} - \frac{1}{4} = \frac{15}{4}$$

> 🔑 **注意**：这里只算了 $\int xy\, dx$，没有 $Q\, dy$ 项（$Q = 0$）。

**例 6.2**（基础，含 $Q\, dy$）计算 $\int_L xy\, dx + (x + y)\, dy$，其中 $L$ 为 $y = x^2$ 从 $O(0, 0)$ 到 $B(2, 4)$。

**步骤 1**：参数 $x$。

**步骤 2**：$dx = dx$，$dy = 2x\, dx$。

**步骤 3**：代入
$$xy\, dx + (x + y)\, dy = x \cdot x^2\, dx + (x + x^2) \cdot 2x\, dx = (x^3 + 2x^2 + 2x^3)\, dx = (3x^3 + 2x^2)\, dx$$

**步骤 4**：$x$ 从 $0$ 到 $2$。
$$\int_L = \int_0^2 (3x^3 + 2x^2)\, dx = \left[\frac{3x^4}{4} + \frac{2x^3}{3}\right]_0^2 = 12 + \frac{16}{3} = \frac{52}{3}$$

**例 6.3**（中等，圆弧）计算 $\int_L -x^2 y\, dx + xy^2\, dy$，其中 $L$ 为圆 $x^2 + y^2 = a^2$ 上从 $A(a, 0)$ 沿逆时针方向到 $B(0, a)$ 的四分之一圆弧。

**步骤 1：选参数**。
圆 → 用参数 $\theta$：$x = a\cos\theta, y = a\sin\theta$。

**步骤 2**：$dx = -a\sin\theta\, d\theta$，$dy = a\cos\theta\, d\theta$。

**步骤 3**：代入
$$-x^2 y\, dx = -(a^2\cos^2\theta)(a\sin\theta)(-a\sin\theta\, d\theta) = a^4 \cos^2\theta \sin^2\theta\, d\theta$$

$$xy^2\, dy = (a\cos\theta)(a^2\sin^2\theta)(a\cos\theta\, d\theta) = a^4 \cos^2\theta \sin^2\theta\, d\theta$$

**步骤 4**：起点 $A(a, 0)$ → $\theta = 0$；终点 $B(0, a)$ → $\theta = \pi/2$。注意题目说"逆时针"，即 $\theta$ 从 $0$ 到 $\pi/2$。

$$\int_L = \int_0^{\pi/2} 2a^4 \cos^2\theta \sin^2\theta\, d\theta = 2a^4 \int_0^{\pi/2} \sin^2\theta \cos^2\theta\, d\theta$$

利用 $\sin^2\theta \cos^2\theta = \dfrac{\sin^2 2\theta}{4}$：
$$= \frac{a^4}{2} \int_0^{\pi/2} \sin^2 2\theta\, d\theta = \frac{a^4}{2} \cdot \frac{\pi}{4} = \frac{\pi a^4}{8}$$

> ⚠️ **方向重要性**：如果题目说"顺时针"，则 $\theta$ 从 $\pi/2$ 到 $0$（或写 $-\int$），结果会差一个**符号**。

**例 6.4**（中等，折线）计算 $\int_L xy\, dx$，其中 $L$ 是从 $A(1, 1)$ 经 $B(1, 2)$ 到 $C(2, 2)$ 的折线段。

**步骤**：折线要分段。
- $L_1$（$A \to B$）：$x = 1, y$ 从 $1$ 到 $2$，$dx = 0$，所以 $\int_{L_1} xy\, dx = 0$。
- $L_2$（$B \to C$）：$y = 2, x$ 从 $1$ 到 $2$，$dy = 0$，$\int_{L_2} xy\, dx = \int_1^2 2x\, dx = x^2\bigg|_1^2 = 3$。

$$\int_L = 0 + 3 = 3$$

> 🔑 **做题经验**：折线 = **分段**，每一段单独算再求和。

### 题目类型

1. 弧段（圆弧、抛物线弧）上的曲线积分
2. 折线段（矩形路径、三角形路径）
3. 空间曲线（螺旋线、空间折线）

## 第三节 曲线积分与路径无关的条件

> 📌 这是**考试重点**——格林公式 + 路径无关 + 全微分方程。

### 一、格林公式

> 这是把**曲线积分**转化为**二重积分**的桥梁。

#### 1. 区域类型

- **单连通区域**：区域内任何闭曲线都能连续缩成一点（如圆盘、矩形区域）。
- **多连通区域**：区域内有"洞"，闭曲线不能缩成一点（如圆环、补了洞的圆盘）。

> 💡 **直观**：单连通 = 区域**没有洞**；多连通 = 区域**有洞**。

#### 2. 格林公式（核心定理）

**定理**：设闭区域 $D$ 由分段光滑的**有向曲线 $L$ 围成**，$P(x, y), Q(x, y)$ 在 $D$ 上具有连续偏导数，则
$$\iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) d\sigma = \oint_L P\, dx + Q\, dy$$

> 🔑 **解释**：
> - **左** = 区域上的二重积分（含偏导）
> - **右** = 沿**边界 $L$ 的正向**的曲线积分（$L$ 是 $D$ 的边界，方向是"区域在左手边"，即逆时针）
> - **正向**：人沿曲线走，区域在左手边（即**逆时针**方向）

> 🔑 **记忆口诀**："**偏 $Q$ 减偏 $P$，叉乘出来 = 沿边逆走**"
> - $\dfrac{\partial Q}{\partial x} - \dfrac{\partial P}{\partial y}$：对 $Q$ 求 $x$ 的偏导，减对 $P$ 求 $y$ 的偏导
> - 写成"$\dfrac{\partial}{\partial x} - \dfrac{\partial}{\partial y}$" × "$\binom{P}{Q}$" = 散度的二维形式

> 📊 看图理解 [[raw/素材/img/fig_6_06_plane_orientation.png]]：椭圆区域的"正向"（逆时针，区域在左手边）和"负向"（顺时针，区域在右手边）。格林公式用**正向**。

#### 3. 例题

**例 6.5**（基础）用格林公式计算 $\oint_L xy^2\, dx + x^2 y\, dy$，其中 $L$ 是圆 $x^2 + y^2 = a^2$ 的正向。

**步骤 1：识别 $P, Q$**。
$P = xy^2$，$Q = x^2 y$。

**步骤 2：算偏导**。
$\dfrac{\partial Q}{\partial x} = 2xy$，$\dfrac{\partial P}{\partial y} = 2xy$。

**步骤 3：套格林公式**。
$D$ = 圆 $x^2 + y^2 \leq a^2$。
$$\oint_L = \iint_D (2xy - 2xy)\, d\sigma = \iint_D 0\, d\sigma = 0$$

> ✅ 答案 $0$。说明 $\oint_L = 0$ 时，曲线积分与路径无关（也满足第三节二的定理）。

**例 6.6**（典型考题）计算 $\oint_L (2x - y + 4)\, dx + (5y + 3x - 6)\, dy$，其中 $L$ 为三顶点 $(0, 0), (3, 0), (3, 2)$ 的三角形正向边界。

**步骤 1**：$P = 2x - y + 4$，$Q = 5y + 3x - 6$。

**步骤 2**：
$\dfrac{\partial Q}{\partial x} = 3$，$\dfrac{\partial P}{\partial y} = -1$。
$\dfrac{\partial Q}{\partial x} - \dfrac{\partial P}{\partial y} = 3 - (-1) = 4$。

**步骤 3**：三角形面积 = $\dfrac{1}{2} \cdot 3 \cdot 2 = 3$。
$$\oint_L = \iint_D 4\, d\sigma = 4 \cdot 3 = 12$$

> 🔑 **做题技巧**：如果 $\dfrac{\partial Q}{\partial x} - \dfrac{\partial P}{\partial y}$ 是**常数**（如本例 $= 4$），直接乘以区域面积即可。

**例 6.7**（中等，做辅助线）计算 $\int_L (e^x \sin y - y^2)\, dx + (e^x \cos y + x)\, dy$，其中 $L$ 是从 $A(1, 0)$ 沿圆 $x^2 + y^2 = 2x$ 上半部分到 $B(0, 0)$ 的弧。

**思路**：$L$ 不是闭合曲线 → 补一条辅助线 $\overline{BA}$（从 $B$ 沿 $x$ 轴到 $A$），让 $L + \overline{BA}$ 构成闭合路径，再套格林公式。

**步骤 1**：$P = e^x \sin y - y^2$，$Q = e^x \cos y + x$。
$\dfrac{\partial Q}{\partial x} = e^x \cos y + 1$，$\dfrac{\partial P}{\partial y} = e^x \cos y - 2y$。
$\dfrac{\partial Q}{\partial x} - \dfrac{\partial P}{\partial y} = 1 + 2y$。

**步骤 2**：圆 $x^2 + y^2 = 2x$ 即 $(x-1)^2 + y^2 = 1$，上半圆，$D$ 是上半圆盘。

$D$ 的面积 $= \dfrac{\pi \cdot 1^2}{2} = \dfrac{\pi}{2}$。
$D$ 的形心 $y$ 坐标 $= \dfrac{4}{3\pi}$（半圆质心到直径距离）→ $\iint_D y\, d\sigma = \dfrac{\pi}{2} \cdot \dfrac{4}{3\pi} = \dfrac{2}{3}$。

$$\iint_D (1 + 2y)\, d\sigma = \frac{\pi}{2} + 2 \cdot \frac{2}{3} = \frac{\pi}{2} + \frac{4}{3}$$

**步骤 3**：算辅助线 $\overline{BA}$（从 $B(0, 0)$ 到 $A(1, 0)$ 沿 $x$ 轴），$y = 0, dy = 0$：
$$\int_{\overline{BA}} = \int_0^1 (e^x \sin 0 - 0)\, dx + 0 = 0$$

**步骤 4**：由格林公式，闭合路径 $L + \overline{BA}$ 取正向（逆时针）= 上面那个二重积分。
$$\oint_{L + \overline{BA}} = \frac{\pi}{2} + \frac{4}{3}$$

但 $L$ 的方向是题目给的（从 $A$ 到 $B$，即沿上半圆从 $A(1, 0)$ 到 $B(0, 0)$，**顺时针**），不是正向。

**判断方向**：上半圆 $D$ 在曲线下方，从 $A(1,0)$ 到 $B(0,0)$ 时区域在右手边 → 是**负向**（顺时针）。

所以 $\int_L = -\oint_{L + \overline{BA}} + \int_{\overline{BA}}$，即
$$\int_L = -\left(\frac{\pi}{2} + \frac{4}{3}\right) - \int_{\overline{BA}}$$

等等，更直接地：$\oint_{L \cup \overline{BA}} = \int_L + \int_{\overline{BA}}$，其中 $L \cup \overline{BA}$ 是正向闭合路径。

但实际 $L$ 是上半圆从 $A$ 到 $B$（顺时针），$\overline{BA}$ 是从 $B$ 到 $A$（沿 $x$ 轴正向）。两者首尾相接，合成闭合路径是**顺时针**。

所以 $\oint_{L \cup \overline{BA}}^{\text{顺时针}} = \int_L + \int_{\overline{BA}} = -\oint_{L \cup \overline{BA}}^{\text{逆时针}} = -\left(\dfrac{\pi}{2} + \dfrac{4}{3}\right)$。

$$\int_L = -\frac{\pi}{2} - \frac{4}{3} - \int_{\overline{BA}} = -\frac{\pi}{2} - \frac{4}{3} - 0 = -\frac{\pi}{2} - \frac{4}{3}$$

> 🔑 **做题技巧**：题目给的不是闭合路径时，**补辅助线**凑闭合，再用格林公式。

**题目类型**：
1. 直接套格林公式（路径是闭合的）
2. 计算二重积分（区域是闭合的）→ 反向用格林公式
3. **补辅助线**构造闭合路径

### 二、平面上曲线积分与路径无关的条件

> 📌 用户要求"每个定理都需要解释 + 一道简单一道复杂例题 + 讲解清晰"

#### 1. 核心定理（四条件等价）

**定理**：设 $P(x, y), Q(x, y)$ 在**单连通区域 $D$** 内具有连续偏导数，则以下四个条件**互相等价**：

1. **环路积分为零**：沿 $D$ 中任一分段光滑的有向闭曲线 $L$，$\oint_L P\, dx + Q\, dy = 0$
2. **与路径无关**：对 $D$ 中任一分段光滑的有向曲线弧 $L$，$\int_L P\, dx + Q\, dy$ 只与 $L$ 的**起点和终点**有关，与路径无关
3. **恰当微分（保守场）**：$P\, dx + Q\, dy$ 是 $D$ 内某个函数 $u(x, y)$ 的**全微分**，即 $du = P\, dx + Q\, dy$
4. **偏导相等**：$\dfrac{\partial P}{\partial y} = \dfrac{\partial Q}{\partial x}$ 在 $D$ 内处处成立

> 🔑 **解释**：
> - 4 个条件是**等价**的——证明一个，其他 3 个自动成立
> - **最重要的判定条件**：4（偏导相等）——最容易验证！
> - **区域必须是单连通**：否则定理失效（多连通区域可能有"洞"里的奇点）

#### 2. 解释：每个条件的含义

| 条件 | 物理意义 |
|------|---------|
| ①环路积分为 0 | 保守场（绕一圈无功）|
| ②与路径无关 | 任意两点间的功只与位置有关 |
| ③有势函数 $u$ | 力是势能的梯度 |
| ④$\partial P/\partial y = \partial Q/\partial x$ | 微分方程可积条件 |

#### 3. 例题

**例 6.8**（简单，验证路径无关）验证 $\int_L xy\, dx + x^2\, dy$ 与路径无关（从 $(0,0)$ 到 $(1,1)$），并计算。

**验证**：$P = xy, Q = x^2$。
$\dfrac{\partial P}{\partial y} = x$，$\dfrac{\partial Q}{\partial x} = 2x$。

二者不相等 → **不是路径无关**！

> ⚠️ **常见错误**：不能看到 $P, Q$ 都连续就判断与路径无关，必须验证 $\partial P/\partial y = \partial Q/\partial x$。

**例 6.9**（简单，求势函数）验证 $\int_L (3x^2 + 4xy)\, dx + (2x^2 + 2y)\, dy$ 与路径无关，并求从 $(0, 0)$ 到 $(1, 1)$ 的积分。

**验证**：$P = 3x^2 + 4xy, Q = 2x^2 + 2y$。
$\dfrac{\partial P}{\partial y} = 4x$，$\dfrac{\partial Q}{\partial x} = 4x$ ✓ → **路径无关**。

**求势函数 $u$**：由 $du = P\, dx + Q\, dy$，有 $\dfrac{\partial u}{\partial x} = 3x^2 + 4xy$。

对 $x$ 积分：$u = x^3 + 2x^2 y + \varphi(y)$（$\varphi(y)$ 是 $y$ 的任意函数）。

由 $\dfrac{\partial u}{\partial y} = 2x^2 + \varphi'(y) = 2x^2 + 2y$ → $\varphi'(y) = 2y$ → $\varphi(y) = y^2 + C$。

所以 $u(x, y) = x^3 + 2x^2 y + y^2$。

**算积分**：
$$\int_L = u(1, 1) - u(0, 0) = (1 + 2 + 1) - 0 = 4$$

> 🔑 **做题套路**：路径无关时，找势函数 $u$ → 积分 = $u(\text{终点}) - u(\text{起点})$。

**例 6.10**（复杂，多连通情形）判断 $\oint_L \dfrac{-y\, dx + x\, dy}{x^2 + y^2}$（$L$ 为绕原点一圈的任意闭曲线）与路径无关性。

**分析**：$\dfrac{\partial P}{\partial y} = \dfrac{-(x^2 + y^2) - (-y)(2y)}{(x^2 + y^2)^2} = \dfrac{y^2 - x^2}{(x^2 + y^2)^2}$
$\dfrac{\partial Q}{\partial x} = \dfrac{(x^2 + y^2) - x(2x)}{(x^2 + y^2)^2} = \dfrac{y^2 - x^2}{(x^2 + y^2)^2}$

二者相等（除了原点外），但**原点是奇点**，区域不是单连通。

**结果**：
- 绕原点一圈时，$\oint_L = 2\pi$（不为 0）
- 不绕原点的闭曲线，$\oint_L = 0$

> 🔑 **重要结论**：多连通区域，**即使偏导相等**，环路积分也不一定为 0。

### 三、全微分方程

**全微分方程**：形如 $P(x, y)\, dx + Q(x, y)\, dy = 0$ 的方程。

**充要条件**：在单连通区域内，$\dfrac{\partial P}{\partial y} = \dfrac{\partial Q}{\partial x}$。

**通解**：$u(x, y) = C$，其中 $u$ 是满足 $du = P\, dx + Q\, dy$ 的势函数。

> 🔑 **做题套路**：
> 1. **验证** $\partial P/\partial y = \partial Q/\partial x$
> 2. **求势函数 $u$**（用线积分或凑微分）
> 3. **通解** = $u(x, y) = C$

#### 例题

**例 6.11**（基础）求 $(2xy + 3)\, dx + (x^2 + 4y)\, dy = 0$ 的通解。

**验证**：$P = 2xy + 3, Q = x^2 + 4y$。
$\dfrac{\partial P}{\partial y} = 2x = \dfrac{\partial Q}{\partial x}$ ✓

**求 $u$**：$\dfrac{\partial u}{\partial x} = 2xy + 3 \implies u = x^2 y + 3x + \varphi(y)$

$\dfrac{\partial u}{\partial y} = x^2 + \varphi'(y) = x^2 + 4y \implies \varphi'(y) = 4y \implies \varphi(y) = 2y^2$

所以 $u = x^2 y + 3x + 2y^2$。

**通解**：$x^2 y + 3x + 2y^2 = C$。

### 积分因子（补充）

如果 $P\, dx + Q\, dy = 0$ **不是全微分方程**（即 $\partial P/\partial y \neq \partial Q/\partial x$），可以**乘一个函数 $\mu(x, y)$**，让 $\mu P\, dx + \mu Q\, dy = 0$ 变成全微分方程。$\mu$ 称为**积分因子**。

**找积分因子的常用方法**：
- $\mu$ 只与 $x$ 有关：$\mu = e^{\int \frac{\partial P/\partial y - \partial Q/\partial x}{Q}\, dx}$
- $\mu$ 只与 $y$ 有关：$\mu = e^{\int \frac{\partial Q/\partial x - \partial P/\partial y}{P}\, dy}$

#### 例题

**例 6.12**（积分因子）求 $\dfrac{dy}{dx} = \dfrac{x + y}{x - y}$ 的通解。

化为微分形式：$(x + y)\, dx - (x - y)\, dy = 0$。即 $P = x + y, Q = -(x - y) = y - x$。

$\dfrac{\partial P}{\partial y} = 1$，$\dfrac{\partial Q}{\partial x} = -1$。不相等 → 不是全微分方程。

**试 $\mu = \mu(x)$**：
$\dfrac{\partial (\mu P)/\partial y - \partial (\mu Q)/\partial x}{Q} = \dfrac{\mu \cdot 1 - \mu \cdot (-1)}{y - x} = \dfrac{2\mu}{y - x}$

要使 $\mu$ 只与 $x$ 有关，需要 $\dfrac{2\mu}{y - x}$ 只是 $x$ 的函数。但分子是 $\mu(x)$，分母含 $y$，不行。

**改试 $\mu = \frac{1}{x^2 + y^2}$**：
$\mu P = \dfrac{x + y}{x^2 + y^2}, \mu Q = \dfrac{y - x}{x^2 + y^2}$

$\dfrac{\partial (\mu P)}{\partial y} = \dfrac{(x^2 + y^2) - (x + y) \cdot 2y}{(x^2 + y^2)^2} = \dfrac{x^2 - y^2 - 2xy}{(x^2 + y^2)^2}$

$\dfrac{\partial (\mu Q)}{\partial x} = \dfrac{-(x^2 + y^2) - (y - x) \cdot 2x}{(x^2 + y^2)^2} = \dfrac{-x^2 - y^2 - 2xy + 2x^2}{(x^2 + y^2)^2} = \dfrac{x^2 - y^2 - 2xy}{(x^2 + y^2)^2}$

二者相等 ✓。

通解为 $\arctan(y/x) + \ln \sqrt{x^2 + y^2} = C$（不展开）。

## 第四节 对坐标的曲面积分的概念

### 一、有向曲面的概念

> 之前算曲面面积（第一型曲面积分）时，曲面**不分正反**。但有些积分（流量、磁场穿过曲面）需要区分正反。

#### 1. 双侧曲面 vs 单侧曲面

**双侧曲面**：能区分出"内侧"和"外侧"两个不同的侧（如球、平面、柱面）。

**单侧曲面**：只有"一侧"，无法区分（如莫比乌斯带）。

> 📌 **考试范围**：通常只考虑**双侧曲面**。

#### 2. 曲面的侧

对于双侧曲面，约定：
- **上侧 / 下侧**（对 $z = f(x, y)$ 形式）：法向量指向上 / 向下
- **前侧 / 后侧**（对 $y = g(x, z)$ 形式）：法向量指向前 / 向后
- **左侧 / 右侧**（对 $x = h(y, z)$ 形式）：法向量指向左 / 向右
- **外侧 / 内侧**（对闭合曲面）：法向量指向外部 / 内部

> 💡 **直观**：曲面是"双面"，选定一侧（法向量方向）后，就叫**有向曲面**。

> 📊 看图理解 [[raw/素材/img/fig_6_01_up_down.png]]：抛物面 $z = x^2 + y^2$ 的"上侧"和"下侧"——法向量方向不同。

![[raw/素材/img/fig_6_01_up_down.png]]

> 📊 看图理解 [[raw/素材/img/fig_6_02_fblr.png]]：椭球面的"前/后"侧和"左/右"侧——投影方向决定怎么命名。

![[raw/素材/img/fig_6_02_fblr.png]]

> 📊 看图理解 [[raw/素材/img/fig_6_03_outside_inside.png]]：闭合曲面（球面）的"外侧"和"内侧"——高斯公式用**外侧**！

![[raw/素材/img/fig_6_03_outside_inside.png]]

### 二、引例 —— 流向曲面一侧的流量

设稳定流速场 $\vec{v}(x, y, z)$，求单位时间通过曲面 $\Sigma$ 的流体质量（流量）。

思路：把 $\Sigma$ 切成小片，每小片上速度近似不变，流量近似为
$$\Delta \Phi \approx \vec{v} \cdot \vec{n}\, \Delta S = v_x \cos\alpha \Delta S + v_y \cos\beta \Delta S + v_z \cos\gamma \Delta S$$

其中 $\vec{n} = (\cos\alpha, \cos\beta, \cos\gamma)$ 是 $\Sigma$ 上某点处的**单位法向量**（指向选定的一侧）。

注意 $\Delta S \cos\gamma$ 是曲面元素 $\Delta S$ 在 $xOy$ 面上的**有向投影**（记作 $dS_{xy}$，可正可负，取决于法向量与 $z$ 轴的夹角）。

求和取极限：
$$\Phi = \lim \sum (v_x \Delta S_{yz} + v_y \Delta S_{zx} + v_z \Delta S_{xy})$$

这就是**对坐标的曲面积分**。

### 三、对坐标的曲面积分的概念

设 $\Sigma$ 是有向光滑曲面，$P(x, y, z), Q(x, y, z), R(x, y, z)$ 在 $\Sigma$ 上有界。类似曲线积分的定义：
$$\iint_\Sigma P\, dydz + Q\, dzdx + R\, dxdy = \lim \sum [P \Delta S_{yz} + Q \Delta S_{zx} + R \Delta S_{xy}]$$

- $P, Q, R$：**被积函数**
- $\Sigma$：**积分曲面**（有方向！）
- $dydz, dzdx, dxdy$：**有向曲面元素在坐标面上的投影**（可正可负！）

**常用形式**：
- $\iint_\Sigma R(x, y, z)\, dxdy$：对 $xOy$ 面的曲面积分
- $\iint_\Sigma P(x, y, z)\, dydz$：对 $yOz$ 面的曲面积分
- $\iint_\Sigma Q(x, y, z)\, dzdx$：对 $zOx$ 面的曲面积分

> 🔑 **与第一型曲面积分的区别**：
> - 第一型 $\iint_\Sigma f\, dS$：$dS > 0$，与方向无关
> - 第二型 $\iint_\Sigma P\, dydz + Q\, dzdx + R\, dxdy$：每个 $dydz$ 等可正可负，与**曲面方向**有关

### 四、对坐标曲面积分的性质

1. **可加性**：$\iint_{\Sigma_1 + \Sigma_2} = \iint_{\Sigma_1} + \iint_{\Sigma_2}$（$\Sigma_1, \Sigma_2$ 同向）
2. **反向变号**：$\iint_{-\Sigma} = -\iint_{\Sigma}$（$\Sigma$ 反向）

## 第五节 对坐标的曲面积分的计算

### 核心思想

> 🔑 **对坐标曲面积分 = 化为二重积分**（对曲面在某个坐标面上的投影区域积分）。

### 公式（投影到 $xOy$ 面）

设 $\Sigma: z = z(x, y)$，$D_{xy}$ 是 $\Sigma$ 在 $xOy$ 面上的**有向投影区域**（保留方向）：

**情形**：$\Sigma$ 取**上侧**（法向量 $\cos\gamma > 0$）：
$$\iint_\Sigma R(x, y, z)\, dxdy = \iint_{D_{xy}} R[x, y, z(x, y)]\, dxdy$$

> 🔑 **关键**：上侧取**正号**！下侧取**负号**：
> $$\iint_{\Sigma_{\text{下侧}}} R\, dxdy = -\iint_{D_{xy}} R[x, y, z(x, y)]\, dxdy$$

> 📊 看图理解 [[raw/素材/img/fig_6_04_projection_sign.png]]：球面分上下两半——上半球外侧 = 上侧（$\cos\gamma > 0$），下半球外侧 = 下侧（$\cos\gamma < 0$），投影正负号不同。

![[raw/素材/img/fig_6_04_projection_sign.png]]

### 公式（完整三种情形）

> 🔑 **记忆口诀**："**曲面向哪一侧投影，就取哪个方向的正负号**"——上侧取正，下侧取负。

| 曲面形式 | 投影到 | 取侧（正） | 取侧（负） |
|---------|-------|----------|----------|
| $z = z(x, y)$ | $D_{xy}$ | 上侧 | 下侧 |
| $x = x(y, z)$ | $D_{yz}$ | 前侧（$\cos\alpha > 0$）| 后侧 |
| $y = y(z, x)$ | $D_{zx}$ | 右侧 | 左侧 |

### 公式（$\iint_\Sigma P\, dydz + Q\, dzdx + R\, dxdy$）

这个最常用的形式需要分别计算三个积分（或用向量形式合并算）。

#### 例题

**例 6.13**（基础）计算 $\iint_\Sigma x\, dxdy$，其中 $\Sigma$ 是平面 $x + y + z = 1$ 在第一卦限部分的上侧。

**步骤 1**：$\Sigma: z = 1 - x - y$，投影到 $xOy$ 面：$D_{xy}: x \geq 0, y \geq 0, x + y \leq 1$。

**步骤 2**：取**上侧** → 用正号。
$$\iint_\Sigma x\, dxdy = \iint_{D_{xy}} x\, dxdy = \int_0^1 x\, dx \int_0^{1-x} dy = \int_0^1 x(1 - x)\, dx = \frac{1}{2} - \frac{1}{3} = \frac{1}{6}$$

**例 6.14**（基础，方向性）计算 $\iint_\Sigma xyz\, dxdy$，其中 $\Sigma$ 是球面 $x^2 + y^2 + z^2 = 1$ 在 $x \geq 0, y \geq 0$ 部分的外侧。

**步骤 1**：球面分上下两部分。
- $\Sigma_1$（上半部分 $z \geq 0$）：$z = \sqrt{1 - x^2 - y^2}$，外侧 = 上侧
- $\Sigma_2$（下半部分 $z \leq 0$）：$z = -\sqrt{1 - x^2 - y^2}$，外侧 = 下侧

**步骤 2**：投影到 $xOy$ 面：$D_{xy}: x^2 + y^2 \leq 1, x \geq 0, y \geq 0$（第一象限单位圆）。

$$\iint_{\Sigma_1} xyz\, dxdy = \iint_{D_{xy}} xy \sqrt{1 - x^2 - y^2}\, dxdy \quad (\text{上侧，正号})$$

$$\iint_{\Sigma_2} xyz\, dxdy = -\iint_{D_{xy}} xy (-\sqrt{1 - x^2 - y^2})\, dxdy = \iint_{D_{xy}} xy \sqrt{1 - x^2 - y^2}\, dxdy \quad (\text{下侧，负号；又因 } z = -\sqrt{\cdots}, \text{ 双重负号})$$

两者相等，相加 = $2 \iint_{D_{xy}} xy \sqrt{1 - x^2 - y^2}\, dxdy$。

**步骤 3**：极坐标 $x = r\cos\theta, y = r\sin\theta$，$\theta \in [0, \pi/2], r \in [0, 1]$：
$$= 2 \int_0^{\pi/2} \int_0^1 r^2 \cos\theta \sin\theta \sqrt{1 - r^2} \cdot r\, drd\theta$$

$$= 2 \int_0^{\pi/2} \cos\theta \sin\theta\, d\theta \cdot \int_0^1 r^3 \sqrt{1 - r^2}\, dr = 2 \cdot \frac{1}{2} \cdot \int_0^1 r^3 \sqrt{1 - r^2}\, dr$$

$$= \int_0^1 r^3 \sqrt{1 - r^2}\, dr$$

令 $u = 1 - r^2, du = -2r\, dr$：$= \dfrac{1}{2}\int_0^1 (1 - u)\sqrt{u}\, du = \dfrac{1}{2}\left[\dfrac{2}{3} - \dfrac{2}{5}\right] = \dfrac{1}{2} \cdot \dfrac{4}{15} = \dfrac{2}{15}$

> 🔑 **做题套路**：
> 1. 看曲面 $\Sigma$ 是不是分块（球面分上下、抛物面分内外）
> 2. 每块分别算，按**侧面方向**取正负号
> 3. 在投影区域上做二重积分

**题目类型**：计算曲面积分的不同侧（上/下、前/后、左/右、外/内）。

## 第六节 高斯公式和斯托克斯公式

### 一、高斯公式（散度定理）

**定理**：设空间有界闭区域 $\Omega$ 由分片光滑的**闭曲面 $\Sigma$** 围成，函数 $P(x, y, z), Q(x, y, z), R(x, y, z)$ 在 $\Omega$ 及 $\Sigma$ 上具有连续偏导数，则
$$\boxed{\iiint_\Omega \left(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\right) dV = ∯_\Sigma P\, dydz + Q\, dzdx + R\, dxdy}$$

其中 $\Sigma$ 取 $\Omega$ 的**整个边界的外侧**。

> 🔑 **解释**：
> - **左** = $\Omega$ 内的三重积分（含散度 $\nabla \cdot \vec{F} = \partial P/\partial x + \partial Q/\partial y + \partial R/\partial z$）
> - **右** = 沿外侧闭曲面的第二型曲面积分
> - **方向**：必须取**外侧**（闭曲面把 $\Omega$ 包在里面）

> 🔑 **记忆口诀**："**散度的体积分 = 外侧的曲面积分**"——散度定理把"内部"和"边界"联系起来。

#### 例题

**例 6.15**（基础）用高斯公式计算 $∯_\Sigma x\, dydz + y\, dzdx + z\, dxdy$，其中 $\Sigma$ 是球面 $x^2 + y^2 + z^2 = R^2$ 的外侧。

**步骤 1**：$P = x, Q = y, R = z$。
$\dfrac{\partial P}{\partial x} + \dfrac{\partial Q}{\partial y} + \dfrac{\partial R}{\partial z} = 1 + 1 + 1 = 3$。

**步骤 2**：套公式，$\Omega$ 是球体。
$$∯_\Sigma = \iiint_\Omega 3\, dV = 3 \cdot \frac{4\pi R^3}{3} = 4\pi R^3$$

**例 6.16**（典型考题）计算 $∯_\Sigma (x^2 - yz)\, dydz + (y^2 - xz)\, dzdx + (z^2 - xy)\, dxdy$，其中 $\Sigma$ 为球面 $x^2 + y^2 + z^2 = a^2$ 的外侧。

**步骤 1**：$P = x^2 - yz, Q = y^2 - xz, R = z^2 - xy$。
$\dfrac{\partial P}{\partial x} = 2x, \dfrac{\partial Q}{\partial y} = 2y, \dfrac{\partial R}{\partial z} = 2z$。
求和 $= 2(x + y + z)$。

**步骤 2**：套公式。
$$∯_\Sigma = \iiint_\Omega 2(x + y + z)\, dV$$

由对称性，$\iiint x\, dV = \iiint y\, dV = \iiint z\, dV = 0$（球对称）。

所以 $∯_\Sigma = 0$。

> 🔑 **做题技巧**：**对称性** + 高斯公式可以秒杀很多看似复杂的曲面积分。

**题目类型**：
1. 直接套高斯公式（闭曲面、外侧）
2. 计算曲面积分（区域是某曲面的某一侧），可能需要**补面**凑闭曲面

### 二、斯托克斯公式

> 📌 用户要求"行列式形式 + 例题 + 右手法则"

**定理**：设 $\Sigma$ 是分片光滑的有向曲面，$L$ 是 $\Sigma$ 的**分段光滑的有向边界曲线**，$P, Q, R$ 在包含 $\Sigma$ 的空间区域内具有连续偏导数，则
$$\iint_\Sigma \left(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}\right) dydz + \left(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}\right) dzdx + \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dxdy = \oint_L P\, dx + Q\, dy + R\, dz$$

> 🔑 **解释**：
> - **左** = 曲面上的第二型曲面积分（含旋度）
> - **右** = 沿边界 $L$ 的曲线积分
> - **方向关系**：$L$ 的方向与 $\Sigma$ 的方向**符合右手定则**

#### 行列式形式（用户要求）

$$\oint_L P\, dx + Q\, dy + R\, dz = \iint_\Sigma \begin{vmatrix} dydz & dzdx & dxdy \\ \dfrac{\partial}{\partial x} & \dfrac{\partial}{\partial y} & \dfrac{\partial}{\partial z} \\ P & Q & R \end{vmatrix}$$

**展开**：
$$= \iint_\Sigma \begin{vmatrix} dydz & dzdx & dxdy \\ \partial_x & \partial_y & \partial_z \\ P & Q & R \end{vmatrix}$$

> 🔑 **记忆口诀**："**第一行是面元，第二行是算子，第三行是被积函数**"

#### 右手法则（用户要求）

> 想象右手握住 $\Sigma$ 的法向量方向，四指指向的方向就是 $L$ 的正向。

```
        Σ (法向量朝上)
        ↑
        │ ╭─────╮
        │ │     │ ← 边界 L 的方向（四指弯曲方向）
        │ ╰─────╯
        │
右手握住 ↑ 方向：
拇指朝上（Σ 法向量）
四指弯曲方向 = L 的正向
```

> 📊 看图理解 [[raw/素材/img/fig_6_05_right_hand.png]]：上半球 + 赤道边界。边界逆时针（从上方看）时，法向量必须朝上（右手系一致）；如果法向量朝下，边界必须顺时针。

![[raw/素材/img/fig_6_05_right_hand.png]]

更具体：法向量 $\vec{n}$ 指向手心时，四指方向 = $L$ 的正方向。

#### 例题

**例 6.17**（基础）用斯托克斯公式计算 $\oint_L -y^2\, dx + x\, dy + z^2\, dz$，其中 $L$ 是平面 $x + y + z = 1$ 与三坐标面的交线（三角形 $ABC$），从原点看逆时针方向。

**步骤 1**：识别 $P, Q, R$。
$P = -y^2, Q = x, R = z^2$

**步骤 2**：算旋度各项。
- $\dfrac{\partial R}{\partial y} - \dfrac{\partial Q}{\partial z} = 0 - 0 = 0$
- $\dfrac{\partial P}{\partial z} - \dfrac{\partial R}{\partial x} = 0 - 0 = 0$
- $\dfrac{\partial Q}{\partial x} - \dfrac{\partial P}{\partial y} = 1 - (-2y) = 1 + 2y$

**步骤 3**：套斯托克斯公式。
$$\oint_L = \iint_\Sigma (1 + 2y)\, dxdy$$

$\Sigma$ 是三角形 $ABC$（在平面 $x + y + z = 1$ 上）。

**投影到 $xOy$ 面**：$D_{xy}: x \geq 0, y \geq 0, x + y \leq 1$。

由右手法则，从原点（$z$ 轴负方向）看逆时针 → 法向量 $\vec{n}$ 指向**上方**（$z$ 正方向）→ 用上侧的正号。

$$\oint_L = \iint_{D_{xy}} (1 + 2y)\, dxdy$$

$$= \int_0^1 dx \int_0^{1-x} (1 + 2y)\, dy = \int_0^1 \left[y + y^2\right]_0^{1-x} dx = \int_0^1 (1 - x + (1-x)^2)\, dx$$

$$= \int_0^1 (1 - x + 1 - 2x + x^2)\, dx = \int_0^1 (2 - 3x + x^2)\, dx = 2 - \frac{3}{2} + \frac{1}{3} = \frac{12 - 9 + 2}{6} = \frac{5}{6}$$

> 🔑 **做题技巧**：用斯托克斯公式把复杂的曲线积分转化为简单的曲面积分。

**例 6.18**（典型考题）计算 $\oint_L (y + z)\, dx + (z + x)\, dy + (x + y)\, dz$，其中 $L$ 是椭圆 $\begin{cases} x^2 + y^2 + z^2 = a^2 \\ x + y + z = 0 \end{cases}$，方向从 $x$ 轴正向看是逆时针。

**步骤 1**：$P = y + z, Q = z + x, R = x + y$。

**步骤 2**：旋度各项。
- $\dfrac{\partial R}{\partial y} - \dfrac{\partial Q}{\partial z} = 1 - 1 = 0$
- $\dfrac{\partial P}{\partial z} - \dfrac{\partial R}{\partial x} = 1 - 1 = 0$
- $\dfrac{\partial Q}{\partial x} - \dfrac{\partial P}{\partial y} = 1 - 1 = 0$

**步骤 3**：旋度 = $(0, 0, 0)$ → $\oint_L = \iint_\Sigma 0 = 0$。

> ✅ **快速判定**：如果旋度 = 0（$\partial Q/\partial x = \partial P/\partial y$ 等），直接得 $\oint_L = 0$。

## 第七节 两类曲线积分、两类曲面积分之间的关系

### 一、两类曲线积分之间的关系

**平面情形**：$\int_L P\, dx + Q\, dy = \int_L (P\cos\alpha + Q\cos\beta)\, ds$

其中 $(\cos\alpha, \cos\beta)$ 是 $L$ 上每点切向量的方向余弦。

**空间情形**：$\int_L P\, dx + Q\, dy + R\, dz = \int_L (P\cos\alpha + Q\cos\beta + R\cos\gamma)\, ds$

> 💡 **理解**：第二型曲线积分 = 把 $dx, dy$ 换成切向量方向余弦 × $ds$。

### 二、两类曲面积分之间的关系

$$\iint_\Sigma P\, dydz + Q\, dzdx + R\, dxdy = \iint_\Sigma (P\cos\alpha + Q\cos\beta + R\cos\gamma)\, dS$$

其中 $(\cos\alpha, \cos\beta, \cos\gamma)$ 是 $\Sigma$ 上每点法向量的方向余弦。

### 三、高斯公式、斯托克斯公式的另一种表示

**高斯公式**（向量形式）：
$$\iiint_\Omega \nabla \cdot \vec{F}\, dV = ∯_\Sigma \vec{F} \cdot \vec{n}\, dS$$
其中 $\vec{F} = (P, Q, R)$，$\nabla \cdot \vec{F} = \dfrac{\partial P}{\partial x} + \dfrac{\partial Q}{\partial y} + \dfrac{\partial R}{\partial z}$，$\vec{n} = (\cos\alpha, \cos\beta, \cos\gamma)$。

**斯托克斯公式**（向量形式）：
$$\iint_\Sigma (\nabla \times \vec{F}) \cdot \vec{n}\, dS = \oint_L \vec{F} \cdot d\vec{r}$$
其中 $\nabla \times \vec{F}$ 是旋度。

## 🎯 总结：六大公式速查

| 公式      | 形式                                                                                 | 适用场景       |
| ------- | ---------------------------------------------------------------------------------- | ---------- |
| 第一型曲面积分 | $\iint_\Sigma f\, dS$                                                              | 曲面面积、曲面质量  |
| 第二型曲面积分 | $\iint_\Sigma P\, dydz + Q\, dzdx + R\, dxdy$                                      | 流量、磁场穿过曲面  |
| 高斯公式    | $\iiint_\Omega \nabla\cdot\vec{F}\,dV = ∯_\Sigma \vec{F}\cdot d\vec{S}$       | 闭曲面外侧的曲面积分 |
| 第一型曲线积分 | $\int_L f\, ds$                                                                    | 弧长、线密度     |
| 第二型曲线积分 | $\int_L P\, dx + Q\, dy$                                                           | 变力做功       |
| 格林公式    | $\iint_D (\partial_x Q - \partial_y P)\, d\sigma = \oint_L P\, dx + Q\, dy$        | 平面闭曲线积分    |
| 斯托克斯公式  | $\iint_\Sigma (\nabla\times\vec{F})\cdot d\vec{S} = \oint_L \vec{F}\cdot d\vec{r}$ | 空间曲线积分     |

> 🔑 **统一思想**：**"内"和"边"互相转化**——
> - 二重积分 ↔ 边界曲线积分（格林）
> - 三重积分 ↔ 边界曲面积分（高斯）
> - 曲面积分 ↔ 边界曲线积分（斯托克斯）

## 🔑 整体做题口诀

> **看到什么 → 想什么 → 用什么公式**

1. **看到曲面积分（闭曲面 + 外侧）** → 高斯公式（化为三重积分）
2. **看到曲线积分（平面闭曲线 + 逆时针）** → 格林公式（化为二重积分）
3. **看到曲线积分（空间 + 闭合）** → 斯托克斯公式（化为曲面积分）
4. **看到曲线积分（开口 + 偏导相等）** → 找势函数（积分 = $u(\text{终点}) - u(\text{起点})$）
5. **看到对弧长 $ds$** → 化为定积分（$\sqrt{x'^2 + y'^2}\, dt$）
6. **看到对坐标 $dx, dy$** → 化为定积分（$dx = x'(t)\, dt$）+ 注意方向
