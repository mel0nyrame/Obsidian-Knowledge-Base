## 第一节 常微分方程的基本概念

### 📖 概念速查

| 概念 | 定义 |
|------|------|
| **常微分方程 (ODE)** | 含未知函数及其导数的方程，自变量只有一个 |
| **阶** | 出现的最高阶导数的阶数 |
| **$n$ 阶线性 ODE** | $y^{(n)} + a_{n-1}(x) y^{(n-1)} + \cdots + a_0(x) y = f(x)$ |
| **$n$ 阶非线性 ODE** | 不满足线性结构的方程 |
| **解 / 积分曲线** | 使方程成为恒等式的函数 $y = \varphi(x)$ |
| **通解 / 通积分** | 含 $n$ 个独立常数（$C_1, C_2, \ldots, C_n$）的解 |
| **特解** | 不含任意常数的解 |
| **初始条件** | $\begin{cases} y(x_0) = y_0 \\ y'(x_0) = y_0' \\ \cdots \end{cases}$（$n$ 个条件定 $n$ 个常数）|
| **解 ODE** | = 求通解 + 用初始条件定特解 |

## 第二节 一阶微分方程及其解法

> 📌 **核心**：看到一阶方程，先**识别题型**再选对应解法。

### 📋 题型速查表

| 题型 | 判定 | 解法 |
|------|------|------|
| 可分离变量 | $y' = f(x) \cdot g(y)$ | 分离 → 积分 |
| 齐次方程 | $y' = f(y/x)$ | 换 $u = y/x$ |
| 可化为齐次 | $y' = f\left(\dfrac{ax + by + c}{a_1 x + b_1 y + c_1}\right)$ | 平移 + 换元 |
| 一阶线性 | $y' + P(x) y = Q(x)$ | 套公式 |
| 伯努利 | $y' + P(x) y = Q(x) y^n$ | 换 $z = y^{1-n}$ |

### 一、可分离变量的微分方程

**形式**：$\dfrac{dy}{dx} = f(x) \cdot g(y)$

**步骤**：
1. 分离：$\dfrac{dy}{g(y)} = f(x)\, dx$
2. 两边积分：$\int \dfrac{dy}{g(y)} = \int f(x)\, dx$
3. 整理为 $F(y) = G(x) + C$

**例 1** 求 $y' = xy$ 的通解。

解：分离：$\dfrac{dy}{y} = x\, dx$
积分：$\ln|y| = \dfrac{x^2}{2} + C_1$
通解：$y = C e^{x^2/2}$（$C = \pm e^{C_1}$）。

> 🔑 **注意**：可分离后 $g(y)$ 分母不能为零。丢失的解 $g(y) = 0$ 需要单独验证（可能含**奇解**）。

### 二、齐次微分方程

**形式**：$\dfrac{dy}{dx} = f\left(\dfrac{y}{x}\right)$

**步骤**：
1. 令 $u = \dfrac{y}{x}$，则 $y = ux$，$\dfrac{dy}{dx} = u + x\dfrac{du}{dx}$
2. 代入得 $u + x\dfrac{du}{dx} = f(u)$ → 化为可分离
3. 分离积分得通解，再回代 $u = y/x$

**例 2** 求 $y' = \dfrac{y}{x} + \tan\dfrac{y}{x}$ 的通解。

解：令 $u = y/x$，则 $u + x u' = u + \tan u$ → $x u' = \tan u$
分离：$\dfrac{du}{\tan u} = \dfrac{dx}{x}$
积分：$-\ln|\cos u| = \ln|x| + C$ → $\cos u = \dfrac{C}{x}$
回代：$\cos\dfrac{y}{x} = \dfrac{C}{x}$。

### 三、可化为齐次的微分方程

**形式**：$\dfrac{dy}{dx} = f\left(\dfrac{ax + by + c}{a_1 x + b_1 y + c_1}\right)$

> 当 $c = c_1 = 0$（即齐次形式）→ 直接处理
> 当 $c, c_1$ 不全为 0：
> 1. 解方程组 $\begin{cases} ax + by + c = 0 \\ a_1 x + b_1 y + c_1 = 0 \end{cases}$ → 得交点 $(x_0, y_0)$
> 2. 平移：$X = x - x_0, Y = y - y_0$
> 3. 化齐次方程求解 → 回代

### 四、一阶线性微分方程（⭐ 必考）

**标准形式**：$\dfrac{dy}{dx} + P(x) y = Q(x)$
- 若 $Q(x) \equiv 0$ → **一阶齐次线性方程** $y' + P(x) y = 0$
- 若 $Q(x) \not\equiv 0$ → **一阶非齐次线性方程**

#### ① 一阶齐次线性方程

$$\dfrac{dy}{dx} + P(x) y = 0 \implies y = C e^{-\int P(x)\, dx}$$

#### ② 一阶非齐次线性方程（套公式！）

> 🔑 **必背通解公式**：
> $$\boxed{y = e^{-\int P(x)\, dx}\left[\int Q(x) e^{\int P(x)\, dx}\, dx + C\right]}$$

**推导思路**（常数变易法）：
1. 齐次通解 $y = C e^{-\int P\, dx}$ 中的 $C$ 换成 $C(x)$
2. 代入原方程解出 $C(x)$
3. 回代得通解

**例 3** 求 $y' + 2xy = x$ 的通解。

解：$P = 2x$，$\int P\, dx = x^2$
$y = e^{-x^2}\left[\int x \cdot e^{x^2}\, dx + C\right] = e^{-x^2}\left[\dfrac{1}{2}e^{x^2} + C\right] = \dfrac{1}{2} + C e^{-x^2}$

**例 4** 求 $y' + \dfrac{y}{x} = \dfrac{\sin x}{x}$ 满足 $y(\pi) = 1$ 的特解。

解：$P = 1/x$，$\int P\, dx = \ln|x|$，$e^{\int P} = x$
$y = \dfrac{1}{x}\left[\int \sin x\, dx + C\right] = \dfrac{-\cos x + C}{x}$
代入 $y(\pi) = 1$：$\dfrac{1 - C}{\pi} = 1$ → $C = 1 - \pi$
特解：$y = \dfrac{1 - \pi - \cos x}{x}$

### 五、伯努利方程

**形式**：$y' + P(x) y = Q(x) y^n$（$n \neq 0, 1$）

**步骤**：
1. 两边除以 $y^n$：$y^{-n} y' + P(x) y^{1-n} = Q(x)$
2. 令 $z = y^{1-n}$，则 $z' = (1-n) y^{-n} y'$
3. 化为一阶线性：$z' + (1-n) P(x) z = (1-n) Q(x)$
4. 套一阶线性公式

## 第三节 微分方程的降阶法

> 📌 **核心思想**：对某些特殊形式的高阶方程，通过**变量替换**降低阶数，化为一阶方程。

### 一、$y^{(n)} = f(x)$ 型

**步骤**：直接**逐次积分 $n$ 次**，每次积分加一个常数。

**例 5** 求 $y''' = \sin x$ 的通解。

解：积分一次：$y'' = -\cos x + C_1$
再积分：$y' = -\sin x + C_1 x + C_2$
再积分：$y = \cos x + \dfrac{C_1}{2} x^2 + C_2 x + C_3$

### 二、$y'' = f(x, y')$ 型（缺 $y$）

**步骤**：
1. 令 $y' = p(x)$，则 $y'' = p'$
2. 代入得 $p' = f(x, p)$（**一阶方程**）
3. 解出 $p(x)$
4. 积分 $y = \int p(x)\, dx + C_2$

**例 6** 求 $y'' - \dfrac{y'}{x} = 0$ 的通解。

解：令 $p = y'$，则 $p' - \dfrac{p}{x} = 0$ → $\dfrac{dp}{p} = \dfrac{dx}{x}$
$p = C_1 x$，即 $y' = C_1 x$
通解：$y = \dfrac{C_1}{2} x^2 + C_2$

### 三、$y'' = f(y, y')$ 型（缺 $x$）

**步骤**：
1. 令 $y' = p(y)$，则 $y'' = p' = \dfrac{dp}{dx} = \dfrac{dp}{dy} \cdot \dfrac{dy}{dx} = p \dfrac{dp}{dy}$
2. 代入得 $p \dfrac{dp}{dy} = f(y, p)$（以 $y$ 为自变量的一阶方程）
3. 解出 $p(y)$
4. 解 $\dfrac{dy}{dx} = p(y)$（可分离）

**例 7** 求 $y'' + (y')^2 = 0$ 的通解（满足 $y(0) = 0, y'(0) = 1$）。

解：令 $p = y'$，则 $p' + p^2 = 0$
$\dfrac{dp}{p^2} = -dx$ → $-\dfrac{1}{p} = -x + C_1$
$p(0) = 1 \implies -\dfrac{1}{1} = C_1$ → $C_1 = -1$
$p = \dfrac{1}{x + 1}$
通解：$y' = \dfrac{1}{x + 1}$ → $y = \ln|x + 1| + C_2$
$y(0) = 0 \implies C_2 = 0$
特解：$y = \ln(x + 1)$

## 第四节 线性微分方程解的结构

### 一、函数组的线性相关 / 线性无关

**线性相关**：$\exists$ 不全为 0 的 $k_1, k_2, \ldots, k_n$ 使 $k_1 y_1 + k_2 y_2 + \cdots + k_n y_n \equiv 0$

**线性无关**：仅当所有 $k_i = 0$ 时恒等式成立。

> 💡 **直观**：
> - **相关** = 这几个函数不是独立的，可以互相表示
> - **无关** = 缺一不可

### 二、二阶齐次线性方程解的结构

**形式**：$y'' + P(x) y' + Q(x) y = 0$

**定理 1（叠加原理）**：若 $y_1, y_2$ 是方程的解，则 $y = C_1 y_1 + C_2 y_2$ 也是解。

**定理 2（通解结构）**：若 $y_1, y_2$ 是方程的**两个线性无关解**，则 $y = C_1 y_1 + C_2 y_2$ 是**通解**。

**定理 3（刘维尔公式 / 降阶法）**：若已知一个非零解 $y_1$，则**与 $y_1$ 线性无关的第二个解**为：
$$y_2 = y_1(x) \int \frac{e^{-\int P(x)\, dx}}{y_1^2(x)}\, dx$$

**判定线性无关**：用 **朗斯基行列式**：
$$W(y_1, y_2) = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} = y_1 y_2' - y_1' y_2$$

- $W(x) \neq 0$ → 线性**无关**（$y_1, y_2$ 构成通解）
- $W(x) \equiv 0$ → 线性**相关**

### 三、二阶非齐次线性方程解的结构

**形式**：$y'' + P(x) y' + Q(x) y = f(x)$

**通解结构**：$y = \underbrace{\bar{y}}_{\text{齐次通解}} + \underbrace{y^*}_{\text{非齐次特解}}$

> 🔑 **核心**：非齐次通解 = 齐次通解 + 非齐次特解

**定理 5（叠加原理）**：若 $y_1^*, y_2^*$ 分别是 $f_1(x), f_2(x)$ 对应的特解，则 $y^* = y_1^* + y_2^*$ 是 $f_1 + f_2$ 对应的特解。

**定理 6（实虚分离）**：若 $y = y_1(x) \pm i y_2(x)$ 是 $f = f_1 \pm i f_2$ 对应的解，则 $y_1, y_2$ 分别是 $f_1, f_2$ 对应的解。

## 第五节 二阶常系数线性微分方程（⭐⭐⭐ 必考大题）

> 📌 **全章最重要**——考试必考大题，几乎年年都有。

### 一、二阶常系数齐次线性方程

**形式**：$y'' + p y' + q y = 0$

#### 🔑 特征方程法（必背）

**步骤**：
1. 写出**特征方程**：$r^2 + pr + q = 0$
2. 求特征根 $r_1, r_2$
3. 按根的情况写通解

**三种情形**（必背）：

| 特征根 | 通解形式 |
|--------|---------|
| **两不等实根** $r_1 \neq r_2$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| **两相等实根** $r_1 = r_2 = r$ | $y = (C_1 + C_2 x) e^{r x}$ |
| **共轭复根** $r = \alpha \pm \beta i$ | $y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$ |

> 🔑 **记忆口诀**：
> - **两不等** → $C_1 e^{r_1 x} + C_2 e^{r_2 x}$
> - **两相等** → 必须**多一个 $x$ 乘子**
> - **复根** → 必须有 $e^{\alpha x}$（实部）+ $\cos, \sin$（虚部）

**例 8** 求 $y'' - 3y' + 2y = 0$ 的通解。

解：特征方程：$r^2 - 3r + 2 = 0 \implies (r-1)(r-2) = 0$
$r_1 = 1, r_2 = 2$（不等实根）
通解：$y = C_1 e^x + C_2 e^{2x}$

**例 9** 求 $y'' - 2y' + y = 0$ 的通解。

解：$r^2 - 2r + 1 = 0 \implies (r-1)^2 = 0$
$r_1 = r_2 = 1$（相等实根）
通解：$y = (C_1 + C_2 x) e^x$

**例 10** 求 $y'' + y = 0$ 的通解。

解：$r^2 + 1 = 0 \implies r = \pm i$，复根：$\alpha = 0, \beta = 1$
通解：$y = C_1 \cos x + C_2 \sin x$

### 二、二阶常系数非齐次线性方程（⭐⭐⭐ 必考）

**形式**：$y'' + p y' + q y = f(x)$

**思路**：通解 = 齐次通解 + 非齐次特解

**关键**：齐次通解用**特征方程法**；特解用**待定系数法**。

#### 🔑 待定系数法

##### 类型 I：$f(x) = e^{\lambda x} P_m(x)$

设特解 $y^* = x^k \cdot Q_m(x) e^{\lambda x}$，其中 $Q_m(x)$ 是与 $P_m(x)$ 同次的多项式（含 $m+1$ 个待定系数），$k$ 取决于 $\lambda$ 与特征根的关系：

| $\lambda$ 与特征根的关系 | $k$ |
|--------------------------|-----|
| $\lambda$ **不是**特征根 | $k = 0$ |
| $\lambda$ 是**单根** | $k = 1$ |
| $\lambda$ 是**重根** | $k = 2$ |

**例 11** 求 $y'' - 3y' + 2y = 2e^x$ 的通解。

解：**齐次部分**（特征方程法）：$y = C_1 e^x + C_2 e^{2x}$（例 8）

**特解**：$\lambda = 1$ 是**单根** → $k = 1$
设 $y^* = A x e^x$
代入化简（前面 ODE 速通页详细推导）→ $-A e^x = 2 e^x$ → $A = -2$
特解：$y^* = -2x e^x$

通解：$y = C_1 e^x + C_2 e^{2x} - 2x e^x$

**例 12** 求 $y'' + y = x^2 + 1$ 的通解。

解：$\lambda = 0$，特征根是 $\pm i$，$\lambda = 0$ 不是特征根 → $k = 0$
设 $y^* = a x^2 + b x + c$
代入：$2a + b x + c + ax^2 + bx + c = x^2 + 1$
比较系数：$a = 1, b = 0, c = 0$
特解：$y^* = x^2 + 1$

通解：$y = C_1 \cos x + C_2 \sin x + x^2 + 1$

##### 类型 II：$f(x) = e^{ax}\left[P_n(x) \cos\beta x + P_m(x) \sin\beta x\right]$

设特解 $y^* = x^k \cdot e^{ax}\left[Q_{\max(n,m)}^{(1)}(x) \cos\beta x + Q_{\max(n,m)}^{(2)}(x) \sin\beta x\right]$

其中 $k$ 取决于 $\alpha + \beta i$ 与特征根的关系（同类型 I）。

**例 13** 求 $y'' + y = \cos x$ 的通解。

解：$\lambda \pm i\beta$ 是 $\pm i$（特征根！）→ $k = 1$
设 $y^* = x(A \cos x + B \sin x)$
代入化简 → $-2A \sin x + 2B \cos x = \cos x$
$2B = 1, -2A = 0 \implies A = 0, B = 1/2$
特解：$y^* = \dfrac{x}{2} \sin x$

通解：$y = C_1 \cos x + C_2 \sin x + \dfrac{x}{2} \sin x$

## 🎯 整体做题套路速查

### 看一眼题型 → 直接套套路

1. **一阶 ODE**：
   - $y' = f(x)g(y)$ → 分离
   - $y' = f(y/x)$ → 换 $u = y/x$
   - $y' + P(x)y = Q(x)$ → 套公式
   - $y' + P(x)y = Q(x)y^n$ → 伯努利（换 $z = y^{1-n}$）

2. **二阶 ODE**（首判断是否常系数）：
   - 常系数 + 齐次 → **特征根法**（3 种情形）
   - 常系数 + 非齐次 → 特征根法 + **待定系数法**
   - 变系数 → 看缺什么（$y$ 或 $x$）
   - $y^{(n)} = f(x)$ → 逐次积分

3. **找通解后**：用初始条件定常数（**代数对消法**）

### 🔑 关键陷阱

> ⚠️ **一阶线性公式**中的 $\int P\, dx$ 别忘指数（$e^{\pm \int P\, dx}$）
> ⚠️ **齐次方程相等根**必须有 $x$ 乘子
> ⚠️ **非齐次待定**：$\lambda$ 是特征根时必须**乘以 $x^k$**（否则消不掉齐次项）
> ⚠️ **朗斯基行列式 $\neq 0$** 才是无关

## 📋 必背公式速记卡

### 一阶 5 类
| 类型 | 通解 / 解法 |
|------|-------------|
| 可分离 | 分离 + 积分 |
| 齐次 | $u = y/x$ 化为一阶 |
| 齐次平移 | 解方程组 + 平移 |
| 一阶线性 | $y = e^{-\int P}[\int Q e^{\int P}\, dx + C]$ |

### 二阶 3 类
| 特征根 | 通解 |
|--------|------|
| $r_1 \neq r_2$ | $C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| $r_1 = r_2 = r$ | $(C_1 + C_2 x) e^{rx}$ |
| $\alpha \pm \beta i$ | $e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$ |

### 待定系数 $k$ 取值
| $\lambda$ 与特征根关系 | $k$ |
