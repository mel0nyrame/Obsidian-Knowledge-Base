## 第一节 空间直角坐标系

### 一、空间直角坐标系

过空间一定点 $O$，作三条相互垂直的数轴，它们都以 $O$ 为原点，且一般取相同的长度单位。这三条数轴分别称为 **$x$ 轴（横轴）**、**$y$ 轴（纵轴）**、**$z$ 轴（竖轴）**，统称**坐标轴**。三个坐标轴的交点 $O$ 称为**坐标原点**。

三条坐标轴的正方向按**右手螺旋定则**确定：右手四指从 $x$ 轴正向弯向 $y$ 轴正向，拇指所指方向即为 $z$ 轴正向。这样的三个坐标轴构成了一个**空间直角坐标系** $Oxyz$。

![[raw/素材/img/fig_01_coordinate_system.png]]

### 二、空间中两点间的距离

设 $M_1(x_1, y_1, z_1)$、$M_2(x_2, y_2, z_2)$ 为空间两点，则它们之间的距离为：

$$d = \left| M_1 M_2 \right| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2}$$

## 第二节 向量及其计算

### 一、向量及其线性计算

#### 1. 向量的概念

**向量**：既有大小又有方向的量称为向量。向量的大小（即长度）称为向量的**模**，记作 $|\vec{a}|$ 或 $|a|$。

**单位向量**：模等于 1 的向量。设 $\vec{a} \neq \vec{0}$，与 $\vec{a}$ 同方向的单位向量记作 $\vec{e}_a$，则 $\vec{e}_a = \dfrac{\vec{a}}{|\vec{a}|}$。

**零向量**：模为 0 的向量，记作 $\vec{0}$ 或 $\mathbf{0}$。零向量的方向是任意的。

**负向量**：与 $\vec{a}$ 模相等、方向相反的向量，记作 $-\vec{a}$。

**向量相等**：模相等且方向相同，记作 $\vec{a} = \vec{b}$。向量与起点无关，仅由其模和方向决定。

#### 2. 向量的线性计算

**加法**

- **平行四边形法则**：两向量 $\vec{a}$、$\vec{b}$ 共起点时，$\vec{a} + \vec{b}$ 是以 $\vec{a}$、$\vec{b}$ 为邻边的平行四边形的对角线向量。
- **三角形法则**：将 $\vec{b}$ 的起点移到 $\vec{a}$ 的终点，则从 $\vec{a}$ 起点到 $\vec{b}$ 终点的向量即为 $\vec{a} + \vec{b}$。

**减法**：$\vec{a} - \vec{b} = \vec{a} + (-\vec{b})$。几何上：共起点时，从 $\vec{b}$ 的终点指向 $\vec{a}$ 的终点的向量就是 $\vec{a} - \vec{b}$。

![[raw/素材/img/fig_02_vector_addition.png]]

![[raw/素材/img/fig_03_vector_subtraction.png]]

**数乘**（数与向量的乘法）：设 $\lambda$ 为实数，$\vec{a}$ 为向量，$\lambda \vec{a}$ 是一个向量，其模为 $|\lambda| \cdot |\vec{a}|$，方向为：
- 当 $\lambda > 0$ 时，与 $\vec{a}$ 同向；
- 当 $\lambda < 0$ 时，与 $\vec{a}$ 反向；
- 当 $\lambda = 0$ 时，为零向量。

设 $\vec{e}_a$ 是与 $\vec{a}$ 同方向的单位向量，则 $\vec{a} = |\vec{a}| \vec{e}_a$，从而 $\vec{e}_a = \dfrac{\vec{a}}{|\vec{a}|}$。

### 二、向量的坐标表示

#### 1. 向量在数轴上的投影

**两个向量的夹角**：设 $\vec{a}$、$\vec{b}$ 为两个非零向量，在空间任取一点 $O$，作 $\overrightarrow{OA} = \vec{a}$、$\overrightarrow{OB} = \vec{b}$，则称 $\theta = \angle AOB$（$0 \leq \theta \leq \pi$）为 $\vec{a}$ 与 $\vec{b}$ 的**夹角**。

**点在数轴上的投影**：设 $P$ 为数轴 $u$ 上一点，过 $P$ 作垂直于 $u$ 的平面，与 $u$ 的交点 $P'$ 称为点 $P$ 在数轴 $u$ 上的**投影**。

**向量在数轴上的投影**：设 $\vec{a} = \overrightarrow{AB}$，$A$、$B$ 在数轴 $u$ 上的投影分别为 $A'$、$B'$，则数轴 $u$ 上的有向线段 $\overrightarrow{A'B'}$ 的值 $A'B'$ 称为 $\vec{a}$ 在数轴 $u$ 上的**投影**，记作 $\mathrm{Prj}_u \vec{a}$。

设 $\vec{a} = \overrightarrow{AB}$，$\alpha$ 为 $\vec{a}$ 与数轴 $u$ 的夹角，则：

$$\mathrm{Prj}_u \vec{a} = |\vec{a}| \cos \alpha$$

**投影的性质（可加性）**：若干个向量的和在某数轴上的投影，等于各向量在该数轴上投影的和：

$$\mathrm{Prj}_u (\vec{a}_1 + \vec{a}_2 + \cdots + \vec{a}_n) = \mathrm{Prj}_u \vec{a}_1 + \mathrm{Prj}_u \vec{a}_2 + \cdots + \mathrm{Prj}_u \vec{a}_n$$

#### 2. 向量的坐标表示

设空间直角坐标系 $Oxyz$，$\vec{i}$、$\vec{j}$、$\vec{k}$ 分别为 $x$、$y$、$z$ 轴正方向上的单位向量，统称为**基本单位向量**。

**向径**：起点固定在原点 $O$、终点为点 $M(x, y, z)$ 的向量 $\vec{r} = \overrightarrow{OM}$ 称为点 $M$ 的**向径**，且

$$\vec{r} = \overrightarrow{OM} = x\vec{i} + y\vec{j} + z\vec{k}$$

上式称为向量 $\vec{r}$ 的**分解式**。

**向量的坐标表示**：设 $\vec{a} = \overrightarrow{M_1 M_2}$，$M_1(x_1, y_1, z_1)$，$M_2(x_2, y_2, z_2)$，则 $\vec{a}$ 在三个坐标轴上的投影分别为

$$a_x = x_2 - x_1, \quad a_y = y_2 - y_1, \quad a_z = z_2 - z_1$$

称 $(a_x, a_y, a_z)$ 为向量 $\vec{a}$ 的**坐标**，记作 $\vec{a} = (a_x, a_y, a_z)$，其分解式为

$$\vec{a} = a_x \vec{i} + a_y \vec{j} + a_z \vec{k}$$

**方向角**：设 $\vec{a}$ 为非零向量，它与 $x$、$y$、$z$ 轴正向的夹角 $\alpha$、$\beta$、$\gamma$ 称为 $\vec{a}$ 的**方向角**。

**方向余弦**：

$$\cos\alpha = \frac{a_x}{|\vec{a}|}, \quad \cos\beta = \frac{a_y}{|\vec{a}|}, \quad \cos\gamma = \frac{a_z}{|\vec{a}|}$$

并满足恒等式：

$$\cos^2 \alpha + \cos^2 \beta + \cos^2 \gamma = 1$$

> **例题**

**例 2.1** 设点 $A(1, 2, 3)$，$B(2, -1, 4)$，求 $\overrightarrow{AB}$ 的模、方向余弦及与 $\overrightarrow{AB}$ 同向的单位向量。

解：
$$\overrightarrow{AB} = (2 - 1,\; -1 - 2,\; 4 - 3) = (1, -3, 1)$$
$$|\overrightarrow{AB}| = \sqrt{1^2 + (-3)^2 + 1^2} = \sqrt{11}$$
方向余弦：
$$\cos\alpha = \frac{1}{\sqrt{11}}, \quad \cos\beta = \frac{-3}{\sqrt{11}}, \quad \cos\gamma = \frac{1}{\sqrt{11}}$$
同向单位向量：
$$\vec{e}_{AB} = \left(\frac{1}{\sqrt{11}},\; \frac{-3}{\sqrt{11}},\; \frac{1}{\sqrt{11}}\right)$$

**例 2.2** 已知 $\vec{a} = (2, -1, 3)$，$\vec{b} = (1, 2, -1)$，求 $\vec{a} + \vec{b}$、$\vec{a} - \vec{b}$、$2\vec{a} - 3\vec{b}$。

解：
$$\vec{a} + \vec{b} = (3,\; 1,\; 2)$$
$$\vec{a} - \vec{b} = (1,\; -3,\; 4)$$
$$2\vec{a} - 3\vec{b} = (4, -2, 6) - (3, 6, -3) = (1, -8, 9)$$

**例 2.3**（中等）已知 $\vec{a}$、$\vec{b}$、$\vec{c}$ 三个向量两两垂直，$|\vec{a}| = 1$，$|\vec{b}| = 2$，$|\vec{c}| = 3$，求 $|3\vec{a} + 2\vec{b} - \vec{c}|$。

解：因为 $\vec{a}$、$\vec{b}$、$\vec{c}$ 两两垂直，故两两正交。利用正交性：
$$|3\vec{a} + 2\vec{b} - \vec{c}|^2 = 9|\vec{a}|^2 + 4|\vec{b}|^2 + |\vec{c}|^2 = 9 + 16 + 9 = 34$$
所以 $|3\vec{a} + 2\vec{b} - \vec{c}| = \sqrt{34}$。

### 三、向量的数量积与向量积

**数量积（点积、内积）**：设 $\vec{a}$、$\vec{b}$ 为两向量，$\theta$ 为其夹角，则

$$\vec{a} \cdot \vec{b} = |\vec{a}| |\vec{b}| \cos\theta$$

结果是一个**数**（标量）。

**运算性质**：
- 交换律：$\vec{a} \cdot \vec{b} = \vec{b} \cdot \vec{a}$
- 分配律：$\vec{a} \cdot (\vec{b} + \vec{c}) = \vec{a} \cdot \vec{b} + \vec{a} \cdot \vec{c}$
- 与数乘结合：$(\lambda \vec{a}) \cdot \vec{b} = \lambda (\vec{a} \cdot \vec{b})$
- $\vec{a} \cdot \vec{a} = |\vec{a}|^2$

**坐标表示**：设 $\vec{a} = (a_x, a_y, a_z)$，$\vec{b} = (b_x, b_y, b_z)$，则

$$\vec{a} \cdot \vec{b} = a_x b_x + a_y b_y + a_z b_z$$

**两向量夹角的余弦公式**：

$$\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}| |\vec{b}|} = \frac{a_x b_x + a_y b_y + a_z b_z}{\sqrt{a_x^2 + a_y^2 + a_z^2}\,\sqrt{b_x^2 + b_y^2 + b_z^2}}$$

**两非零向量垂直的充要条件**：

$$\vec{a} \perp \vec{b} \iff \vec{a} \cdot \vec{b} = 0 \iff a_x b_x + a_y b_y + a_z b_z = 0$$

**向量积（叉积、外积）**：设 $\vec{a}$、$\vec{b}$ 为两向量，$\theta$ 为其夹角，$\vec{a} \times \vec{b}$ 是一个**向量**，定义为：
- **模**：$|\vec{a} \times \vec{b}| = |\vec{a}| |\vec{b}| \sin\theta$
- **方向**：垂直于 $\vec{a}$、$\vec{b}$ 所在平面，方向按**右手螺旋定则**（四指从 $\vec{a}$ 弯向 $\vec{b}$，拇指方向即为 $\vec{a} \times \vec{b}$ 的方向）。

**运算性质**：
- 反交换律：$\vec{a} \times \vec{b} = -(\vec{b} \times \vec{a})$
- 分配律：$\vec{a} \times (\vec{b} + \vec{c}) = \vec{a} \times \vec{b} + \vec{a} \times \vec{c}$
- 与数乘结合：$(\lambda \vec{a}) \times \vec{b} = \lambda (\vec{a} \times \vec{b})$
- $\vec{a} \times \vec{a} = \vec{0}$

**坐标表示（行列式）**：设 $\vec{a} = (a_x, a_y, a_z)$，$\vec{b} = (b_x, b_y, b_z)$，则

$$\vec{a} \times \vec{b} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ a_x & a_y & a_z \\ b_x & b_y & b_z \end{vmatrix} = (a_y b_z - a_z b_y)\vec{i} - (a_x b_z - a_z b_x)\vec{j} + (a_x b_y - a_y b_x)\vec{k}$$

即 $\vec{a} \times \vec{b} = (a_y b_z - a_z b_y,\; a_z b_x - a_x b_z,\; a_x b_y - a_y b_x)$。

**两非零向量平行的充要条件**：

$$\vec{a} \parallel \vec{b} \iff \vec{a} \times \vec{b} = \vec{0} \iff \frac{a_x}{b_x} = \frac{a_y}{b_y} = \frac{a_z}{b_z}$$

![[raw/素材/img/fig_05_cross_product.png]]

> **例题**

**例 2.4** 设 $\vec{a} = (2, 1, -1)$，$\vec{b} = (1, -1, 2)$，求 $\vec{a} \cdot \vec{b}$、$\vec{a} \times \vec{b}$，并求 $\vec{a}$、$\vec{b}$ 夹角的余弦。

解：
$$\vec{a} \cdot \vec{b} = 2 \times 1 + 1 \times (-1) + (-1) \times 2 = 2 - 1 - 2 = -1$$
$$\vec{a} \times \vec{b} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 2 & 1 & -1 \\ 1 & -1 & 2 \end{vmatrix}$$
$$= (1 \times 2 - (-1) \times (-1))\vec{i} - (2 \times 2 - (-1) \times 1)\vec{j} + (2 \times (-1) - 1 \times 1)\vec{k}$$
$$= (2 - 1)\vec{i} - (4 + 1)\vec{j} + (-2 - 1)\vec{k} = (1, -5, -3)$$

$$|\vec{a}| = \sqrt{4 + 1 + 1} = \sqrt{6}, \quad |\vec{b}| = \sqrt{1 + 1 + 4} = \sqrt{6}$$
$$\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}|\,|\vec{b}|} = \frac{-1}{6} = -\frac{1}{6}$$

**例 2.5**（中等）已知 $\vec{a} = (1, 2, 3)$，$\vec{b} = (2, 4, 6)$，判断 $\vec{a}$ 与 $\vec{b}$ 是否平行。

解法 1（分量比值）：观察 $\vec{b} = 2\vec{a}$，分量成比例 $\dfrac{1}{2} = \dfrac{2}{4} = \dfrac{3}{6} = \dfrac{1}{2}$，所以 $\vec{a} \parallel \vec{b}$。

解法 2（叉积）：
$$\vec{a} \times \vec{b} = (2 \cdot 6 - 3 \cdot 4,\; 3 \cdot 2 - 1 \cdot 6,\; 1 \cdot 4 - 2 \cdot 2) = (0, 0, 0) = \vec{0}$$
所以 $\vec{a} \parallel \vec{b}$。

**例 2.6**（稍难）已知三角形 $ABC$ 的顶点 $A(1, 2, 3)$、$B(3, 4, 5)$、$C(2, 4, 7)$，求 $\triangle ABC$ 的面积。

解：
$$\overrightarrow{AB} = (2, 2, 2), \quad \overrightarrow{AC} = (1, 2, 4)$$
$$\overrightarrow{AB} \times \overrightarrow{AC} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 2 & 2 & 2 \\ 1 & 2 & 4 \end{vmatrix} = (8 - 4)\vec{i} - (8 - 2)\vec{j} + (4 - 2)\vec{k} = (4, -6, 2)$$
$$S = \frac{1}{2}\left|\overrightarrow{AB} \times \overrightarrow{AC}\right| = \frac{1}{2}\sqrt{16 + 36 + 4} = \frac{1}{2}\sqrt{56} = \sqrt{14}$$

## 第三节 空间平面与空间直线

### 一、空间平面的方程

**法向量**：垂直于平面的非零向量，记作 $\vec{n} = (A, B, C)$。

**平面的点法式方程**：设平面 $\Pi$ 过点 $M_0(x_0, y_0, z_0)$，法向量 $\vec{n} = (A, B, C)$，则平面方程为：

$$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$$

**平面的一般方程**：将点法式方程展开，得到

$$Ax + By + Cz + D = 0$$

其中 $\vec{n} = (A, B, C)$ 为平面的法向量。

**平面的截距式方程**：当平面与三个坐标轴都相交且不平行时，设三个交点分别为 $(a, 0, 0)$、$(0, b, 0)$、$(0, 0, c)$（$abc \neq 0$），则平面方程为：

$$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$$

其中 $a$、$b$、$c$ 称为平面在三个坐标轴上的**截距**。

**特殊位置的平面方程**：

- **过原点的平面**：$D = 0$（$Ax + By + Cz = 0$）
- **平行于某坐标轴的平面**：法向量在该轴上的分量为 0。
  - 平行于 $x$ 轴：$A = 0$，方程 $By + Cz + D = 0$
  - 平行于 $y$ 轴：$B = 0$，方程 $Ax + Cz + D = 0$
  - 平行于 $z$ 轴：$C = 0$，方程 $Ax + By + D = 0$
- **过某坐标轴的平面**：缺一个变量且常数项 $D = 0$。
  - 过 $x$ 轴：$By + Cz = 0$
  - 过 $y$ 轴：$Ax + Cz = 0$
  - 过 $z$ 轴：$Ax + By = 0$
- **垂直于某坐标轴的平面**：法向量沿该轴方向。
  - 垂直于 $x$ 轴：$x = x_0$（即 $x - x_0 = 0$）
  - 垂直于 $y$ 轴：$y = y_0$
  - 垂直于 $z$ 轴：$z = z_0$

### 二、空间直线的方程

**直线的一般方程（交面式）**：直线可看作两个不平行的平面的交线：

$$\begin{cases} A_1 x + B_1 y + C_1 z + D_1 = 0 \\ A_2 x + B_2 y + C_2 z + D_2 = 0 \end{cases}$$

**直线的对称式方程（标准式、点向式）**：设直线过点 $M_0(x_0, y_0, z_0)$，方向向量 $\vec{s} = (m, n, p)$，则

$$\frac{x - x_0}{m} = \frac{y - y_0}{n} = \frac{z - z_0}{p}$$

要求 $m$、$n$、$p$ 不全为 0；当某分母为 0 时，按对应分子也为 0 理解（如 $m = 0$ 时，$x = x_0$）。

**直线的参数方程**：

$$\begin{cases} x = x_0 + m t \\ y = y_0 + n t \\ z = z_0 + p t \end{cases} \quad (t \in \mathbb{R})$$

其中 $t$ 为参数。

**直线的两点式方程**：设直线过 $M_1(x_1, y_1, z_1)$、$M_2(x_2, y_2, z_2)$ 两点，则

$$\frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1} = \frac{z - z_1}{z_2 - z_1}$$

![[raw/素材/img/fig_12_line.png]]

> **例题**

**例 2.7** 求过点 $M_0(1, -2, 3)$ 且垂直于平面 $2x - y + 3z + 5 = 0$ 的直线方程。

解：直线的方向向量取为平面的法向量 $\vec{s} = (2, -1, 3)$。
对称式：
$$\frac{x - 1}{2} = \frac{y + 2}{-1} = \frac{z - 3}{3}$$
参数式：
$$\begin{cases} x = 1 + 2t \\ y = -2 - t \\ z = 3 + 3t \end{cases}$$

**例 2.8** 求过点 $A(1, 0, -1)$、$B(2, 1, 0)$ 的直线方程。

解：方向向量 $\vec{s} = \overrightarrow{AB} = (1, 1, 1)$。
对称式：
$$\frac{x - 1}{1} = \frac{y}{1} = \frac{z + 1}{1}$$
参数式：
$$\begin{cases} x = 1 + t \\ y = t \\ z = -1 + t \end{cases}$$

**例 2.9**（稍难）求过点 $M_0(2, -1, 3)$ 且与直线 $\dfrac{x - 1}{2} = \dfrac{y}{-1} = \dfrac{z + 2}{3}$ 垂直相交的直线方程。

解：已知直线 $L$ 的方向向量 $\vec{s}_1 = (2, -1, 3)$，设所求直线 $L'$ 与 $L$ 的交点为 $A$。$L$ 的参数式为 $(1 + 2t,\; -t,\; -2 + 3t)$。设 $A(1 + 2t, -t, -2 + 3t)$，则

$$\overrightarrow{M_0 A} = (1 + 2t - 2,\; -t + 1,\; -2 + 3t - 3) = (2t - 1,\; 1 - t,\; 3t - 5)$$

由 $\overrightarrow{M_0 A} \perp \vec{s}_1$（即 $\overrightarrow{M_0 A} \cdot \vec{s}_1 = 0$）：

$$2(2t - 1) + (-1)(1 - t) + 3(3t - 5) = 0$$
$$4t - 2 - 1 + t + 9t - 15 = 0 \implies 14t - 18 = 0 \implies t = \frac{9}{7}$$

代入 $L$ 的参数式得 $A\!\left(\dfrac{31}{7},\; -\dfrac{9}{7},\; \dfrac{13}{7}\right)$。

方向向量 $\overrightarrow{M_0 A} = \left(\dfrac{17}{7},\; -\dfrac{2}{7},\; -\dfrac{2}{7}\right)$，可取 $\vec{s} = (17, -2, -2)$。

所求直线方程：
$$\frac{x - 2}{17} = \frac{y + 1}{-2} = \frac{z - 3}{-2}$$

### 三、平面与直线的位置关系

**两平面的夹角**：设两平面 $\Pi_1: A_1 x + B_1 y + C_1 z + D_1 = 0$，$\Pi_2: A_2 x + B_2 y + C_2 z + D_2 = 0$，其法向量分别为 $\vec{n}_1 = (A_1, B_1, C_1)$，$\vec{n}_2 = (A_2, B_2, C_2)$，夹角 $\theta$（取锐角或直角）满足：

$$\cos\theta = \frac{|\vec{n}_1 \cdot \vec{n}_2|}{|\vec{n}_1|\,|\vec{n}_2|} = \frac{|A_1 A_2 + B_1 B_2 + C_1 C_2|}{\sqrt{A_1^2 + B_1^2 + C_1^2}\,\sqrt{A_2^2 + B_2^2 + C_2^2}}$$

**两平面垂直的充要条件**：

$$\Pi_1 \perp \Pi_2 \iff \vec{n}_1 \cdot \vec{n}_2 = 0 \iff A_1 A_2 + B_1 B_2 + C_1 C_2 = 0$$

**两平面平行的充要条件**：

$$\Pi_1 \parallel \Pi_2 \iff \vec{n}_1 \parallel \vec{n}_2 \iff \frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2}$$

**两条直线的夹角**：设两直线方向向量分别为 $\vec{s}_1 = (m_1, n_1, p_1)$，$\vec{s}_2 = (m_2, n_2, p_2)$，夹角 $\theta$（取锐角或直角）：

$$\cos\theta = \frac{|\vec{s}_1 \cdot \vec{s}_2|}{|\vec{s}_1|\,|\vec{s}_2|}$$

**两直线垂直的充要条件**：

$$L_1 \perp L_2 \iff \vec{s}_1 \cdot \vec{s}_2 = 0 \iff m_1 m_2 + n_1 n_2 + p_1 p_2 = 0$$

**两直线平行的充要条件**：

$$L_1 \parallel L_2 \iff \vec{s}_1 \parallel \vec{s}_2 \iff \frac{m_1}{m_2} = \frac{n_1}{n_2} = \frac{p_1}{p_2}$$

**直线与平面平行的充要条件**：设直线 $L$ 方向向量 $\vec{s} = (m, n, p)$，平面 $\Pi$ 法向量 $\vec{n} = (A, B, C)$：

$$L \parallel \Pi \iff \vec{s} \cdot \vec{n} = 0 \iff Am + Bn + Cp = 0$$

**直线与平面垂直的充要条件**：

$$L \perp \Pi \iff \vec{s} \parallel \vec{n} \iff \frac{m}{A} = \frac{n}{B} = \frac{p}{C}$$

## 第四节 空间曲面与空间曲线

### 一、空间曲面的方程

一般地，曲面 $S$ 与三元方程 $F(x, y, z) = 0$ 满足：
1. 曲面 $S$ 上任一点的坐标都满足该方程；
2. 不在曲面 $S$ 上的点的坐标都不满足该方程。

则称 $F(x, y, z) = 0$ 为曲面 $S$ 的**方程**，曲面 $S$ 为方程 $F(x, y, z) = 0$ 的**图形**。

#### 1. 球面方程

**球面的一般方程**：以 $M_0(x_0, y_0, z_0)$ 为球心、$R$ 为半径的球面方程为：

$$(x - x_0)^2 + (y - y_0)^2 + (z - z_0)^2 = R^2$$

展开后的形式为：

$$x^2 + y^2 + z^2 + Dx + Ey + Fz + G = 0$$

> 识别特征：$x^2$、$y^2$、$z^2$ 系数相等且没有 $xy$、$yz$、$zx$ 等交叉项。

![[raw/素材/img/fig_06_sphere.png]]

#### 2. 柱面

**柱面**：设有一条定直线 $l$，另有一条曲线 $C$，沿 $C$ 平行于 $l$ 移动，移动时所形成的曲面称为**柱面**。
- **准线**：原曲线 $C$
- **母线**：沿准线移动的定直线 $l$

> **重要说明**：柱面方程中**缺哪个变量**，该柱面就**平行于哪个坐标轴**（或者说其母线方向沿那个坐标轴）。例如方程 $x^2 + y^2 = 1$ 中缺少 $z$，所以它表示以 $xOy$ 面上的圆 $x^2 + y^2 = 1$ 为准线、母线平行于 $z$ 轴的**圆柱面**。

常见二次柱面（母线均平行 $z$ 轴）：

- **圆柱面**：$x^2 + y^2 = R^2$
- **椭圆柱面**：$\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$
- **双曲柱面**：$\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$
- **抛物柱面**：$y^2 = 2px$

![[raw/素材/img/fig_10_cylinder.png]]

### 二、旋转曲面

**旋转曲面**：一条曲线 $C$ 绕一条定直线 $l$ 旋转一周所形成的曲面。
- **轴**：定直线 $l$ 称为旋转轴
- **纬圆**：曲线 $C$ 上任一点绕轴旋转都得到一个圆，称为**纬圆**
- **母线**：原曲线 $C$ 本身

**旋转曲面的方程**：设 $C$ 在 $yOz$ 面上，方程为 $f(y, z) = 0$，绕 $z$ 轴旋转，则旋转曲面的方程为：

$$f\!\left(\pm\sqrt{x^2 + y^2},\; z\right) = 0$$

> 记忆方法：绕哪个轴旋转，哪个轴的坐标保持不变；另一个坐标用 $\pm\sqrt{x^2 + y^2}$ 替换。

### 三、二次曲面举例

**二次曲面**：三元二次方程 $F(x, y, z) = 0$ 所表示的曲面。

**截痕法**：用坐标平面或平行于坐标平面的平面去截曲面，通过观察所得交线（截痕）的形状来研究曲面的形状。

#### 1. 椭球面

**方程**：

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1 \quad (a, b, c > 0)$$

**半轴**：$a$、$b$、$c$ 分别为椭球面在 $x$、$y$、$z$ 轴正方向的半轴长。椭球面被包围在长方体 $|x| \leq a$、$|y| \leq b$、$|z| \leq c$ 内。

特殊情形：
- $a = b = c$ 时为**球面**
- $a = b \neq c$ 时为**旋转椭球面**

![[raw/素材/img/fig_07_ellipsoid.png]]

#### 2. 双曲面

**(1) 单叶双曲面**

**方程**：

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$$

特征：方程左侧两项为正、一项为负。

**(2) 双叶双曲面**

**方程**：

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = -1 \quad \text{（即 } -\frac{x^2}{a^2} - \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1\text{）}$$

特征：方程左侧两项为负、一项为正。

![[raw/素材/img/fig_08_hyperboloids.png]]

#### 3. 抛物面

**(1) 椭圆抛物面**

**方程**：

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = z \quad (a, b > 0)$$

**(2) 双曲抛物面（马鞍面）**

**方程**：

$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = z \quad (a, b > 0)$$

![[raw/素材/img/fig_09_paraboloids.png]]

### 四、空间曲线的方程

**空间曲线的一般方程**：空间曲线可看作两个曲面的交线：

$$\begin{cases} F_1(x, y, z) = 0 \\ F_2(x, y, z) = 0 \end{cases}$$

**空间曲线的参数方程**：

$$\begin{cases} x = x(t) \\ y = y(t) \\ z = z(t) \end{cases}$$

**空间曲线在坐标面上的投影**：
1. 由曲线方程消去 $z$ 得 $F(x, y) = 0$，这是曲线在 $xOy$ 面上的**投影柱面**方程。
2. 投影柱面与 $xOy$ 面的交线即为曲线在 $xOy$ 面上的**投影曲线**。
3. 类似地可求在 $xOz$、$yOz$ 面上的投影。

![[raw/素材/img/fig_13_projection.png]]

> **例题**

**例 2.10** 指出下列方程所表示的曲面：
(1) $x^2 + y^2 + z^2 - 2x + 4y = 0$
(2) $x^2 + y^2 - z^2 = 1$

解：
(1) 配方：$(x - 1)^2 - 1 + (y + 2)^2 - 4 + z^2 = 0$，即 $(x - 1)^2 + (y + 2)^2 + z^2 = 5$。这是以 $(1, -2, 0)$ 为球心、$\sqrt{5}$ 为半径的**球面**。

(2) 用截痕法分析：
- 平行于 $xOy$ 面的平面 $z = h$ 截得 $x^2 + y^2 = 1 + h^2$（圆，半径 $\sqrt{1 + h^2}$，$\forall h$ 都成立）
- 平行于 $xOz$ 面的平面 $y = 0$ 截得 $x^2 - z^2 = 1$（双曲线）
- 平行于 $yOz$ 面的平面 $x = 0$ 截得 $y^2 - z^2 = 1$（双曲线）

这是**单叶旋转双曲面**（绕 $z$ 轴旋转 $x^2 - z^2 = 1$ 而成）。

**例 2.11**（中等）求曲线 $\Gamma: \begin{cases} x^2 + y^2 + z^2 = 1 \\ x^2 + y^2 - z^2 = 0 \end{cases}$ 在 $xOy$ 面上的投影曲线方程。

解：两式相加得 $2(x^2 + y^2) = 1$，即 $x^2 + y^2 = \dfrac{1}{2}$。相减得 $2z^2 = 1$，即 $z = \pm\dfrac{1}{\sqrt{2}}$。

$\Gamma$ 在 $xOy$ 面上的**投影柱面**方程为 $x^2 + y^2 = \dfrac{1}{2}$，在 $xOy$ 面上的**投影曲线**为：

$$\begin{cases} x^2 + y^2 = \dfrac{1}{2} \\ z = 0 \end{cases}$$
