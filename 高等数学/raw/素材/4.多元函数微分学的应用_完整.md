## 第一节 空间曲线的切线和法平面

### 一、空间曲线的切线（直观理解）

设空间曲线 $\Gamma$ 过点 $P_0$，在 $\Gamma$ 上 $P_0$ 附近另取一点 $P$，作**割线** $P_0 P$。当 $P$ 沿 $\Gamma$ 趋近于 $P_0$ 时，割线 $P_0 P$ 的**极限位置**就是曲线在 $P_0$ 处的**切线**。

切线的**方向向量**称为曲线在 $P_0$ 处的**切向量**。过 $P_0$ 且**垂直于切线**的平面称为曲线在 $P_0$ 处的**法平面**。

> 🔑 **做题套路**：求切线 → 先求**切向量**（曲线的"运动方向"）；求法平面 → 切向量就是法平面的**法向量**。

### 二、求切线与法平面的两种情形

#### 情形 1：曲线为**参数方程**

设曲线 $\Gamma: x = x(t), y = y(t), z = z(t)$，$t_0$ 对应点 $P_0(x_0, y_0, z_0)$。

**切向量**（曲线在 $P_0$ 处的"运动方向"）：
$$\vec{s} = \big(x'(t_0),\ y'(t_0),\ z'(t_0)\big)$$

**切线方程**（对称式 / 点向式）：
$$\frac{x - x_0}{x'(t_0)} = \frac{y - y_0}{y'(t_0)} = \frac{z - z_0}{z'(t_0)}$$

**法平面方程**（用切向量作为法向量）：
$$x'(t_0)(x - x_0) + y'(t_0)(y - y_0) + z'(t_0)(z - z_0) = 0$$

#### 情形 2：曲线为**两个曲面的交线**（一般方程）

设曲线 $\Gamma: \begin{cases} F(x, y, z) = 0 \\ G(x, y, z) = 0 \end{cases}$，$P_0(x_0, y_0, z_0)$。

每个方程都对应一个曲面的**法向量**：
$$\vec{n}_F = (F_x, F_y, F_z)\big|_{P_0}, \quad \vec{n}_G = (G_x, G_y, G_z)\big|_{P_0}$$

切向量 = **两个法向量的叉积**（因为两个法向量都垂直于切线，它们的叉积就沿着切线方向）：
$$\vec{s} = \vec{n}_F \times \vec{n}_G = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ F_x & F_y & F_z \\ G_x & G_y & G_z \end{vmatrix}\bigg|_{P_0}$$

即 $\vec{s} = (F_y G_z - F_z G_y,\ F_z G_x - F_x G_z,\ F_x G_y - F_y G_x)\big|_{P_0}$。

**切线方程**：
$$\frac{x - x_0}{F_y G_z - F_z G_y}\bigg|_{P_0} = \frac{y - y_0}{F_z G_x - F_x G_z}\bigg|_{P_0} = \frac{z - z_0}{F_x G_y - F_y G_x}\bigg|_{P_0}$$

**法平面方程**：
$$s_x(x - x_0) + s_y(y - y_0) + s_z(z - z_0) = 0$$

> 🔑 **两种情形对比**（你之前提示"两种形式不一样"）：
> - **参数式**：切向量 = $(x'(t_0), y'(t_0), z'(t_0))$
> - **一般方程**：切向量 = 两个法向量的**叉积**（行列式）

> **例题**

**例 4.1** 求螺旋线 $\Gamma: x = \cos t, y = \sin t, z = t$ 在 $t = \pi/2$ 处的切线方程与法平面方程。

解：$t_0 = \pi/2$，$P_0 = (0, 1, \pi/2)$。
$x'(t) = -\sin t \Rightarrow x'(\pi/2) = -1$
$y'(t) = \cos t \Rightarrow y'(\pi/2) = 0$
$z'(t) = 1$

切向量 $\vec{s} = (-1, 0, 1)$。

**切线方程**：
$$\frac{x - 0}{-1} = \frac{y - 1}{0} = \frac{z - \pi/2}{1}$$

（中间分母为 0 表示 $y = 1$）

**法平面方程**：
$$-1(x - 0) + 0(y - 1) + 1(z - \pi/2) = 0 \implies -x + z = \pi/2$$

**例 4.2** 求曲线 $\Gamma: \begin{cases} x^2 + y^2 + z^2 = 4 \\ x + y + z = 0 \end{cases}$ 在点 $P_0(1, 1, -\sqrt{2})$ 处的切线方程与法平面方程。

解：$F = x^2 + y^2 + z^2 - 4$，$G = x + y + z$。
$F_x = 2x, F_y = 2y, F_z = 2z$；$G_x = G_y = G_z = 1$。

在 $P_0(1, 1, -\sqrt{2})$：
$\vec{n}_F = (2, 2, -2\sqrt{2})$，$\vec{n}_G = (1, 1, 1)$。

切向量 = 叉积：
$$\vec{s} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 2 & 2 & -2\sqrt{2} \\ 1 & 1 & 1 \end{vmatrix} = (2 + 2\sqrt{2})\vec{i} - (2 + 2\sqrt{2})\vec{j} + 0\vec{k} = (2 + 2\sqrt{2})(1, -1, 0)$$

取 $\vec{s} = (1, -1, 0)$。

**切线方程**：
$$\frac{x - 1}{1} = \frac{y - 1}{-1} = \frac{z + \sqrt{2}}{0}$$

（中间分母为 0 表示 $z = -\sqrt{2}$）

**法平面方程**：
$$1(x - 1) - 1(y - 1) + 0(z + \sqrt{2}) = 0 \implies x - y = 0$$

## 第二节 空间曲面的切平面与法线

> 📌 本节用到很多偏导和雅可比式，我一步一步讲，按第一次听讲的方式展开。

### 一、直观理解

设曲面 $\Sigma: F(x, y, z) = 0$，$M_0(x_0, y_0, z_0)$ 是曲面上一点。在曲面上过 $M_0$ 任意画一条曲线 $\Gamma$（曲线 $\Gamma$ 是 $\Sigma$ 和某个平面的交线）。这条曲线在 $M_0$ 处的**切线方向**取决于 $\Gamma$ 的方向——但有一个神奇的事实：

> 🔑 **所有过 $M_0$ 的曲线在该点的切线都落在同一个平面上**。这个平面就是曲面在 $M_0$ 处的**切平面**。过 $M_0$ 垂直于切平面的直线称为曲面在该点的**法线**。

### 二、推导（为什么切平面的法向量 = 梯度）

设曲面 $\Sigma: F(x, y, z) = 0$，过 $M_0$ 有一条曲线 $\Gamma$，其参数方程为
$$x = x(t),\ y = y(t),\ z = z(t)$$

由于 $\Gamma$ 在曲面上，将参数方程代入曲面方程：
$$F[x(t),\ y(t),\ z(t)] \equiv 0 \quad (\text{恒等式，对所有 } t)$$

> 这里用恒等号 $\equiv$（不是 $=$）是因为：曲线 $\Gamma$ 在曲面上 $\Rightarrow$ 曲线上**每一个**点都满足曲面方程，所以代入后对**所有** $t$ 都等于 0。简单记：`=` 是"这一刻相等"，`≡` 是"永远相等"。后面会对该恒等式两边对 $t$ 求导，所以恒等号语义上更准确。

对 $t$ 求导（链式法则）：
$$F_x \cdot x'(t) + F_y \cdot y'(t) + F_z \cdot z'(t) = 0$$

引入两个向量：
- **梯度**（曲面在 $M_0$ 处的法向量候选）：$\vec{n} = (F_x, F_y, F_z)\big|_{M_0}$
- **切向量**（曲线 $\Gamma$ 在 $M_0$ 处的运动方向）：$\vec{s} = (x'(t_0), y'(t_0), z'(t_0))$

上面的式子其实就是**向量点积**：
$$\vec{n} \cdot \vec{s} = 0$$

> 这说明 $\vec{n}$ 垂直于曲面上**任意**过 $M_0$ 的曲线在 $M_0$ 处的切线。即 $\vec{n}$ 是所有这些切线所在平面的法向量。

**结论**：$\vec{n} = (F_x, F_y, F_z)\big|_{M_0}$ 就是曲面在 $M_0$ 处的**法向量**。

### 三、求切平面与法线的三种情形

#### 情形 1：曲面为**一般方程** $F(x, y, z) = 0$

要求 $F$ 在 $M_0$ 处具有连续偏导数，且 $(F_x, F_y, F_z)\big|_{M_0}$ 不全为零。

**法向量**（梯度）：
$$\vec{n} = (F_x, F_y, F_z)\big|_{M_0}$$

**切平面方程**（用点法式）：
$$F_x(M_0)(x - x_0) + F_y(M_0)(y - y_0) + F_z(M_0)(z - z_0) = 0$$

**法线方程**（以法向量为方向）：
$$\frac{x - x_0}{F_x(M_0)} = \frac{y - y_0}{F_y(M_0)} = \frac{z - z_0}{F_z(M_0)}$$

#### 情形 2：曲面为**显式方程** $z = f(x, y)$

把显式改写为一般式：$F(x, y, z) = f(x, y) - z = 0$。

则 $F_x = f_x$，$F_y = f_y$，$F_z = -1$。

**法向量**：
$$\vec{n} = (f_x, f_y, -1)\big|_{M_0}$$

**切平面方程**：
$$f_x(x_0, y_0)(x - x_0) + f_y(x_0, y_0)(y - y_0) - (z - z_0) = 0$$

**法线方程**：
$$\frac{x - x_0}{f_x(x_0, y_0)} = \frac{y - y_0}{f_y(x_0, y_0)} = \frac{z - z_0}{-1}$$

> 🔑 **做题套路（显式）**：$z = f(x,y)$ → 法向量 = $(f_x, f_y, -1)$。**别忘了 $-1$**！

#### 情形 3：曲面为**参数方程** $x = x(u, v), y = y(u, v), z = z(u, v)$

把曲面看成一个向量值函数：
$$\vec{r}(u, v) = (x(u, v), y(u, v), z(u, v))$$

求两个偏导（沿 $u$ 方向、$v$ 方向的切向量）：
$$\vec{r}_u = (x_u, y_u, z_u), \quad \vec{r}_v = (x_v, y_v, z_v)$$

**法向量** = 两个切向量的**叉积**：
$$\vec{n} = \vec{r}_u \times \vec{r}_v = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ x_u & y_u & z_u \\ x_v & y_v & z_v \end{vmatrix}\bigg|_{M_0}$$

**切平面方程**：
$$n_x(x - x_0) + n_y(y - y_0) + n_z(z - z_0) = 0$$

**法线方程**：
$$\frac{x - x_0}{n_x} = \frac{y - y_0}{n_y} = \frac{z - z_0}{n_z}$$

> 🔑 **三种情形统一公式**：
>
> | 方程形式 | 法向量 $\vec{n}$ |
> |---------|---------------|
> | 一般式 $F(x,y,z) = 0$ | $(F_x, F_y, F_z)$ |
> | 显式 $z = f(x, y)$ | $(f_x, f_y, -1)$ |
> | 参数式 | $\vec{r}_u \times \vec{r}_v$ |
>
> 找到法向量后，切平面用点法式，法线用对称式。

> **例题**

**例 4.3** 求球面 $x^2 + y^2 + z^2 = 14$ 在点 $M_0(1, 2, 3)$ 处的切平面方程与法线方程。

解：$F = x^2 + y^2 + z^2 - 14$。
$F_x = 2x, F_y = 2y, F_z = 2z$。

在 $M_0(1, 2, 3)$：$\vec{n} = (2, 4, 6)$，化简为 $(1, 2, 3)$。

**切平面**：$1(x - 1) + 2(y - 2) + 3(z - 3) = 0 \implies x + 2y + 3z = 14$。

**法线**：$\dfrac{x - 1}{1} = \dfrac{y - 2}{2} = \dfrac{z - 3}{3}$。

**例 4.4** 求曲面 $z = x^2 + y^2$ 在点 $M_0(1, 1, 2)$ 处的切平面方程与法线方程。

解：$f_x = 2x, f_y = 2y$。
在 $M_0(1, 1, 2)$：$f_x = 2, f_y = 2$，法向量 $\vec{n} = (2, 2, -1)$。

**切平面**：$2(x - 1) + 2(y - 1) - (z - 2) = 0 \implies 2x + 2y - z = 2$。

**法线**：$\dfrac{x - 1}{2} = \dfrac{y - 1}{2} = \dfrac{z - 2}{-1}$。

## 第三节 方向导数

### 一、方向导数的定义

$\mathbb{R}^3$ 中任一**单位向量** $\vec{e}_l = (l_1, l_2, l_3)$（即 $|\vec{e}_l| = 1$）称为一个**方向**。

设 $u = f(x, y, z)$ 在 $P_0(x_0, y_0, z_0)$ 的某邻域内有定义，$\vec{e}_l$ 为单位方向向量。若
$$\lim_{t \to 0^+} \frac{f(x_0 + t l_1, y_0 + t l_2, z_0 + t l_3) - f(x_0, y_0, z_0)}{t}$$
存在，则称其为 $f$ 在 $P_0$ 处沿 $\vec{e}_l$ 方向的**方向导数**，记作
$$\frac{\partial f}{\partial \vec{e}_l}\bigg|_{P_0} \quad \text{或} \quad \frac{\partial f}{\partial l}\bigg|_{P_0}$$

### 二、方向导数公式

> 🔑 **定理**（必背）：若 $u = f(x, y, z)$ 在 $P_0(x_0, y_0, z_0)$ 处**可微**，则该函数在 $P_0$ 处沿**任意方向** $\vec{e}_l = (\cos\alpha, \cos\beta, \cos\gamma)$ 的方向导数都存在，且
> $$\boxed{\frac{\partial f}{\partial \vec{e}_l}\bigg|_{P_0} = f_x \cos\alpha + f_y \cos\beta + f_z \cos\gamma}$$

其中 $\cos\alpha, \cos\beta, \cos\gamma$ 是 $\vec{e}_l$ 的**方向余弦**（因为 $|\vec{e}_l| = 1$，三个分量直接就是余弦值）。

> 🔑 **做题套路**（三步走）：
> 1. **单位化**：如果题目给的 $\vec{l}$ 不是单位向量，先算 $\vec{e}_l = \vec{l} / |\vec{l}|$，三个分量就是 $\cos\alpha, \cos\beta, \cos\gamma$
> 2. **求偏导**：算 $f_x, f_y, f_z$ 在 $P_0$ 处的值
> 3. **套公式**：代入 $\dfrac{\partial f}{\partial \vec{e}_l} = f_x \cos\alpha + f_y \cos\beta + f_z \cos\gamma$

> ⚠️ **注意方向**：如果题目说"沿从 $A$ 到 $B$ 的方向"，方向向量是 $\overrightarrow{AB} = B - A$，然后**再单位化**。

**题目类型**：
1. 已知 $\vec{l}$（任意方向）→ **先单位化** → 再套公式
2. 已知 $\vec{l}$ 的方向角 → 直接 $\cos$ 各值 → 套公式
3. 已知**起点和终点**（从 $A$ 到 $B$）→ 方向向量 = $\overrightarrow{AB}$ → 单位化 → 套公式

> **例题**

**例 4.5** 求 $f(x, y, z) = xy + yz + zx$ 在点 $P_0(1, 1, 1)$ 处沿 $\vec{l} = (1, 2, 2)$ 方向的方向导数。

解：
**第一步：单位化** $\vec{l}$。
$|\vec{l}| = \sqrt{1 + 4 + 4} = 3$
$\vec{e}_l = (1/3, 2/3, 2/3)$，即 $\cos\alpha = 1/3, \cos\beta = 2/3, \cos\gamma = 2/3$

**第二步：求偏导**。
$f_x = y + z$，在 $P_0$ 处 $= 1 + 1 = 2$
$f_y = x + z$，在 $P_0$ 处 $= 1 + 1 = 2$
$f_z = y + x$，在 $P_0$ 处 $= 1 + 1 = 2$

**第三步：套公式**。
$$\frac{\partial f}{\partial \vec{l}}\bigg|_{P_0} = 2 \cdot \frac{1}{3} + 2 \cdot \frac{2}{3} + 2 \cdot \frac{2}{3} = \frac{2 + 4 + 4}{3} = \frac{10}{3}$$

**例 4.6** 求 $f(x, y) = x^2 + y^2$ 在点 $P_0(1, 1)$ 处沿**从 $A(1, 1)$ 到 $B(2, 3)$** 的方向的方向导数。

解：
方向向量 $\overrightarrow{AB} = (2 - 1, 3 - 1) = (1, 2)$。
$|\overrightarrow{AB}| = \sqrt{1 + 4} = \sqrt{5}$
$\cos\alpha = 1/\sqrt{5}, \cos\beta = 2/\sqrt{5}$。

偏导：$f_x = 2x = 2$（在 $P_0$），$f_y = 2y = 2$。

$$\frac{\partial f}{\partial \vec{l}}\bigg|_{P_0} = 2 \cdot \frac{1}{\sqrt{5}} + 2 \cdot \frac{2}{\sqrt{5}} = \frac{6}{\sqrt{5}} = \frac{6\sqrt{5}}{5}$$

## 第四节 多元函数的极值及其求法

### 一、多元函数的极值及最值

**极值定义**：

设 $f(x, y)$ 在点 $P_0(x_0, y_0)$ 的某邻域内有定义：

- **极大值**：$\exists\, \delta > 0$，当 $0 < \sqrt{(x - x_0)^2 + (y - y_0)^2} < \delta$ 时，$f(x, y) < f(x_0, y_0)$
- **极小值**：$\exists\, \delta > 0$，当 $0 < \sqrt{(x - x_0)^2 + (y - y_0)^2} < \delta$ 时，$f(x, y) > f(x_0, y_0)$
- **极值点**：取极大值或极小值的点
- **极值**：极大值或极小值

#### 必要条件（费马引理）

> 🔑 **定理**：若 $f(x, y)$ 在区域 $D$ 内可微，则 $f$ 在 $D$ 内一点 $P_0(x_0, y_0)$ 处**取极值**的必要条件是
> $$f_x(x_0, y_0) = 0, \quad f_y(x_0, y_0) = 0$$

称满足 $f_x = f_y = 0$ 的点为**驻点**（也称稳定点）。

> ⚠️ **驻点不一定是极值点**！反例：$f(x, y) = xy$，在 $(0, 0)$ 处 $f_x = f_y = 0$（是驻点），但 $(0, 0)$ 不是极值点（沿 $y = x$ 是正的，沿 $y = -x$ 是负的）。

#### 充分条件

> 🔑 **定理**：设 $f(x, y)$ 在驻点 $P_0(x_0, y_0)$ 处具有**二阶连续偏导数**，令
> $$A = f_{xx}(x_0, y_0), \quad B = f_{xy}(x_0, y_0), \quad C = f_{yy}(x_0, y_0)$$
> 记判别式 $\Delta = AC - B^2$，则：

| $\Delta$ 符号 | 结论 |
|--------------|------|
| $\Delta > 0$ | $A > 0$ → 极小值；$A < 0$ → 极大值 |
| $\Delta < 0$ | 不是极值（鞍点） |
| $\Delta = 0$ | **不能确定**（需另作讨论） |

> 🔑 **口诀**："**$\Delta > 0$ 看 $A$ 定大小（$A$ 正极小、$A$ 负极大）；$\Delta < 0$ 不是极值；$\Delta = 0$ 不讨论**"

#### 最值的求法

求连续函数 $f(x, y)$ 在**有界闭区域** $D$ 上的最值：

1. 求 $D$ **内部**所有驻点 → 算出函数值
2. 求 $D$ **边界**上的最值（化为一元函数或用拉格朗日乘数法）
3. 比较所有候选值 → 取**最大、最小**

> **例题**

**例 4.7** 求 $f(x, y) = x^3 - 3x + y^2 - 2y$ 的极值。

解：
**第一步：求驻点**。
$f_x = 3x^2 - 3 = 0 \implies x = \pm 1$
$f_y = 2y - 2 = 0 \implies y = 1$

驻点：$(1, 1)$、$(-1, 1)$。

**第二步：算二阶偏导**。
$f_{xx} = 6x, f_{xy} = 0, f_{yy} = 2$

**第三步：判别**。

驻点 $(1, 1)$：$A = 6, B = 0, C = 2$ → $\Delta = 12 > 0$，且 $A > 0$ → **极小值**，$f(1, 1) = 1 - 3 + 1 - 2 = -3$。

驻点 $(-1, 1)$：$A = -6, B = 0, C = 2$ → $\Delta = -12 < 0$ → **不是极值**。

**结论**：$f$ 在 $(1, 1)$ 处取极小值 $-3$。

**例 4.8** 求 $f(x, y) = xy(1 - x - y)$ 在三角形区域 $D: x \geq 0, y \geq 0, x + y \leq 1$ 上的最值。

解：
**第一步：内部驻点**。
$f_x = y(1 - 2x - y) = 0$
$f_y = x(1 - x - 2y) = 0$
- $x = 0$ 或 $y = 0$：在边界上
- 内部 $x, y > 0$：解 $\begin{cases} 1 - 2x - y = 0 \\ 1 - x - 2y = 0 \end{cases} \implies x = y = 1/3$
  $f(1/3, 1/3) = (1/3)(1/3)(1/3) = 1/27$

**第二步：边界**。
- $x = 0$：$f = 0$
- $y = 0$：$f = 0$
- $x + y = 1$（令 $y = 1 - x, x \in [0, 1]$）：$f = x(1-x)(1 - 1) = 0$

**第三步：比较**。
最大值为 $1/27$（在 $(1/3, 1/3)$），最小值为 $0$（在整个边界上）。

### 二、条件极值（拉格朗日乘数法）

**条件极值问题**：在约束条件 $\varphi(x, y) = 0$ 下，求 $f(x, y)$ 的极值。

**拉格朗日乘数法**：

构造**拉格朗日函数**：
$$L(x, y, \lambda) = f(x, y) + \lambda\, \varphi(x, y)$$

其中 $\lambda$ 称为**拉格朗日乘数**。

> 🔑 **做题套路**（三步走）：
> 1. 构造 $L = f + \lambda \varphi$
> 2. 解方程组：
> $$\begin{cases} L_x = f_x + \lambda \varphi_x = 0 \\ L_y = f_y + \lambda \varphi_y = 0 \\ L_\lambda = \varphi(x, y) = 0 \end{cases}$$
> 3. 求出 $(x, y, \lambda)$ → 代入 $f$ 得候选极值

> ⚠️ **多个约束条件**怎么办？例如两个约束 $\varphi_1 = 0, \varphi_2 = 0$：
> 构造 $L = f + \lambda_1 \varphi_1 + \lambda_2 \varphi_2$，对应三个偏导方程组（$L_x = L_y = L_z = 0$）加两个约束方程。

**题目类型**：求函数极值、求面被平面截后的最短距离。

> **例题**

**例 4.9** 求 $f(x, y) = x + y$ 在约束 $x^2 + y^2 = 1$ 下的极值。

解：构造 $L = x + y + \lambda(x^2 + y^2 - 1)$。

方程组：
$$\begin{cases} L_x = 1 + 2\lambda x = 0 \\ L_y = 1 + 2\lambda y = 0 \\ x^2 + y^2 - 1 = 0 \end{cases}$$

前两式相减得 $x = y$，代入约束 $2x^2 = 1 \implies x = y = \pm\dfrac{1}{\sqrt{2}}$。

候选值：
- $f\left(\dfrac{1}{\sqrt{2}}, \dfrac{1}{\sqrt{2}}\right) = \sqrt{2}$（**最大值**）
- $f\left(-\dfrac{1}{\sqrt{2}}, -\dfrac{1}{\sqrt{2}}\right) = -\sqrt{2}$（**最小值**）

**例 4.10** 求原点到平面 $x + y + z = 1$ 的最短距离。

解：距离 $d = \sqrt{x^2 + y^2 + z^2}$。求 $d$ 最小 等价于 求 $f = x^2 + y^2 + z^2$ 在约束 $x + y + z - 1 = 0$ 下最小。

构造 $L = x^2 + y^2 + z^2 + \lambda(x + y + z - 1)$。

方程组：
$$\begin{cases} L_x = 2x + \lambda = 0 \\ L_y = 2y + \lambda = 0 \\ L_z = 2z + \lambda = 0 \\ x + y + z - 1 = 0 \end{cases}$$

前 3 式相加减得 $x = y = z = -\lambda/2$，代入约束 $-\dfrac{3\lambda}{2} = 1 \implies \lambda = -\dfrac{2}{3}$。
得 $x = y = z = 1/3$。

最短距离 $d = \sqrt{3 \cdot (1/3)^2} = \sqrt{1/3} = \dfrac{\sqrt{3}}{3}$。
