## ⭐ 三大公式（必考，分值最高）

### 格林公式（平面）
$$\iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) d\sigma = \oint_L P\, dx + Q\, dy$$
**方向**：$L$ 取**正向**（逆时针，区域在左手边）
**适用**：平面**闭曲线** + 偏导易算

### 高斯公式（散度定理）
$$\iiint_\Omega \left(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\right) dV = \oiint_\Sigma P\, dydz + Q\, dzdx + R\, dxdy$$
**方向**：$\Sigma$ 取**外侧**
**适用**：**闭曲面** + 外侧

### 斯托克斯公式（旋度定理）
$$\oint_L P\, dx + Q\, dy + R\, dz = \iint_\Sigma \begin{vmatrix} dydz & dzdx & dxdy \\ \partial_x & \partial_y & \partial_z \\ P & Q & R \end{vmatrix}$$
**方向**：**右手系**（握住法向量，四指方向 = L 正方向）
**适用**：**空间曲线积分**

> 🔑 **统一思想**：内 ↔ 边 互相转化

## ⭐ 重积分速查

### 二重积分（X 形 / Y 形 / 极坐标）
| 形式 | 公式 |
|------|------|
| X 形 | $\int_a^b dx \int_{y_1(x)}^{y_2(x)} f\, dy$ |
| Y 形 | $\int_c^d dy \int_{x_1(y)}^{x_2(y)} f\, dx$ |
| 极坐标 | $\int_\alpha^\beta d\theta \int_0^{r(\theta)} f \cdot r\, dr$ |

> **看到圆 → 极坐标**，**别忘了乘 $r$**
> **看到 $e^{y^2}, \sin y/y$** → 必须**先对 $x$ 积分**

### 三重积分（柱面 / 球面）
| 坐标 | $dV$ | 何时用 |
|------|------|--------|
| 直角 | $dxdydz$ | 一般 |
| 柱面 | $r\, drd\theta\, dz$ | 含圆柱、圆锥 |
| 球面 | $\rho^2 \sin\varphi\, d\rho\, d\varphi\, d\theta$ | 含球 |

> **球面坐标的 $\varphi$** = 从 $z$ 轴往下量的极角（$0 \leq \varphi \leq \pi$）

## ⭐ 无穷级数

### 4 种判别法速记
| 判别法 | 适用 | 公式 |
|--------|------|------|
| 比较 | 通项是分数 | $\lim u_n/v_n = k$ |
| 比值 | 含阶乘、$a^n/n$ | $\lim u_{n+1}/u_n = \rho$ |
| 根式 | 含 $n$ 次方 | $\lim \sqrt[n]{u_n} = \rho$ |
| 莱布尼茨 | 交错级数 | $u_n$ 单调 $\to 0$ |

> **$\rho < 1$ 收，$\rho > 1$ 散，$\rho = 1$ 失效**

### $p$ 级数 / 几何级数（必背参照系）
- $p$ 级数 $\sum 1/n^p$：$p > 1$ 收，$p \leq 1$ 散
- 几何级数 $\sum aq^{n-1}$：$|q| < 1$ 收，$|q| \geq 1$ 散

### 收敛半径
$$R = \lim_{n \to \infty} \left|\frac{a_n}{a_{n+1}}\right|$$
**端点必须单独判！**（最常考）

### 8 个必背麦克劳林展开
$$\frac{1}{1-x} = \sum_{n=0}^\infty x^n \quad |x| < 1$$
$$e^x = \sum \frac{x^n}{n!} \quad |x| < \infty$$
$$\sin x = \sum \frac{(-1)^n}{(2n+1)!} x^{2n+1} \quad |x| < \infty$$
$$\cos x = \sum \frac{(-1)^n}{(2n)!} x^{2n} \quad |x| < \infty$$
$$\ln(1+x) = \sum \frac{(-1)^{n-1}}{n} x^n \quad -1 < x \leq 1$$
$$\frac{1}{1+x^2} = \sum (-1)^n x^{2n} \quad |x| < 1$$
$$\arctan x = \sum \frac{(-1)^n}{2n+1} x^{2n+1} \quad |x| \leq 1$$
$$(1+x)^\alpha = \sum \binom{\alpha}{n} x^n \quad |x| < 1$$

## ⭐ ODE 速通（4 类核心）

### ① 可分离 $\dfrac{dy}{dx} = f(x) \cdot g(y)$
- 分离：$\dfrac{dy}{g(y)} = f(x)\, dx$
- 两边积分

### ② 一阶线性 $y' + p(x) y = q(x)$
- 通解公式：$y = e^{-\int p\, dx}\left[\int q(x) e^{\int p\, dx}\, dx + C\right]$

### ③ 二阶常系数齐次 $y'' + py' + qy = 0$
- 特征方程 $r^2 + pr + q = 0$
- 两不等实根 $r_1 \neq r_2$：$y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$
- 两相等实根 $r_1 = r_2$：$y = (C_1 + C_2 x) e^{r_1 x}$
- 复数根 $\alpha \pm \beta i$：$y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$

### ④ 二阶常系数非齐次 $y'' + py' + qy = f(x)$
- 通解 = 齐次通解 + 非齐次特解 $y^*$
- $f(x) = P_n(x) e^{\lambda x}$（$\lambda$ 不是特征根）：$y^* = Q_n(x) e^{\lambda x}$
- $f(x) = P_n(x) e^{\lambda x} \cos\omega x$ 或 $\sin\omega x$：$y^* = e^{\lambda x}[Q_n^{(1)}(x) \cos\omega x + Q_n^{(2)}(x) \sin\omega x]$

## ⭐ 几何应用速记

### 曲线切向量
- 参数式：$\vec{s} = (x', y', z')$
- 一般式：$\vec{s} = \nabla F \times \nabla G$

### 曲面法向量 = 梯度
$$\vec{n} = \nabla F = (F_x, F_y, F_z)$$

### 方向导数公式
$$\frac{\partial f}{\partial \vec{e}_l} = f_x \cos\alpha + f_y \cos\beta + f_z \cos\gamma$$

### 拉格朗日乘数法
$$L = f + \lambda \varphi, \quad L_x = L_y = L_z = 0, \quad \varphi = 0$$
