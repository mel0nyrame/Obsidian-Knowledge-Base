# 函数展开成幂级数

## 📌 两种方法

### 直接法
按 $a_n = f^{(n)}(0)/n!$ 算系数（繁琐）

### 间接法（必会）
利用已知展开式 + 幂级数的运算（加减、换元、求导、积分）

## 🔑 必背基础展开式

| 函数 | 展开式 | 收敛域 |
|------|-------|--------|
| $\dfrac{1}{1-x}$ | $\sum_{n=0}^\infty x^n$ | $\|x\| < 1$ |
| $e^x$ | $\sum \dfrac{x^n}{n!}$ | $\|x\| < +\infty$ |
| $\sin x$ | $\sum \dfrac{(-1)^n}{(2n+1)!} x^{2n+1}$ | $\|x\| < +\infty$ |
| $\cos x$ | $\sum \dfrac{(-1)^n}{(2n)!} x^{2n}$ | $\|x\| < +\infty$ |
| $\ln(1+x)$ | $\sum \dfrac{(-1)^{n-1}}{n} x^n$ | $-1 < x \leq 1$ |
| $\arctan x$ | $\sum \dfrac{(-1)^n}{2n+1} x^{2n+1}$ | $\|x\| \leq 1$ |

## 🔑 二项展开式（补充）

$$(1+x)^\alpha = \sum_{n=0}^\infty \binom{\alpha}{n} x^n, \quad \binom{\alpha}{n} = \frac{\alpha(\alpha-1)\cdots(\alpha-n+1)}{n!}$$

（$\alpha$ 任意实数，收敛域 $|x| < 1$，端点 $x = \pm 1$ 单独讨论）

## 💡 做题套路

> 🔑 **间接法三步走**：
> 1. **变形**：把目标函数变形（凑已知展开式的形式）
> 2. **套公式**：代入已知展开式
> 3. **整理**：换元 / 求导 / 积分得到目标

## 🔑 常见套路

| 目标 | 套路 |
|------|------|
| $\dfrac{1}{1 - ax}$ | 用 $\dfrac{1}{1-x} = \sum x^n$，换 $x \to ax$ |
| $\ln(1+x)$ | 从 $\dfrac{1}{1+x} = \sum (-1)^n x^n$，积分 |
| $\arctan x$ | 从 $\dfrac{1}{1+x^2} = \sum (-1)^n x^{2n}$，积分 |
| 含 $x^2$ 的偶函数 | 套 $\sin, \cos, \arctan$ |

## 🔑 应用

1. **近似计算**：算 $e, \sin x, \ln 2$ 等的近似值（用前几项）
2. **微分方程求解**：设 $y = \sum a_n x^n$ → 代入 → 比较系数

## ⚠️ 易错点

- 直接法很繁琐，**优先用间接法**
- 收敛域端点要单独判
