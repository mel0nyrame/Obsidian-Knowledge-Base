# 多元复合函数求导（链式法则）

## 📌 定义

> 把一元链式法则 $\dfrac{dy}{dx} = \dfrac{dy}{du} \cdot \dfrac{du}{dx}$ 推广到多元。

## 🔑 关键公式

### 情形 1：$z = f(u, v)$，$u = u(x, y), v = v(x, y)$

$$\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u} \frac{\partial u}{\partial x} + \frac{\partial f}{\partial v} \frac{\partial v}{\partial x}$$

$$\frac{\partial z}{\partial y} = \frac{\partial f}{\partial u} \frac{\partial u}{\partial y} + \frac{\partial f}{\partial v} \frac{\partial v}{\partial y}$$

### 情形 2：$z = f(x, y)$，$x = x(s, t), y = y(s, t)$

$$\frac{\partial z}{\partial s} = \frac{\partial f}{\partial x} \frac{\partial x}{\partial s} + \frac{\partial f}{\partial y} \frac{\partial y}{\partial s}$$

### 情形 3：一元复合 $z = f(u, v)$，$u = u(t), v = v(t)$

$$\frac{dz}{dt} = \frac{\partial f}{\partial u} \frac{du}{dt} + \frac{\partial f}{\partial v} \frac{dv}{dt}$$

## 💡 做题套路

> 🔑 **画变量关系图**（树状）：箭头方向 = 因果关系
> 🔑 **沿链相乘，沿并相加**：每个路径相乘，所有路径相加

```
z → u → x    （链 1）
z → v → x    （链 2）
       → y   （链 3）
       → y   （链 4）
∂z/∂x = (∂f/∂u)(∂u/∂x) + (∂f/∂v)(∂v/∂x)
```

## ⚠️ 易错点

- 全导数 $\dfrac{dz}{dt}$ 用 $d$，偏导 $\dfrac{\partial z}{\partial x}$ 用 $\partial$
- 抽象函数求偏导时，**不要把 $u, v$ 写成 $x, y$**（保持记号区分）
