# 对坐标的曲线积分（第二型曲线积分）

## 📌 定义

> **对坐标的曲线积分** = $\int_L P\, dx + Q\, dy$
> 物理意义：变力 $\vec{F} = (P, Q)$ 沿曲线 $L$ 做的**功**

## 🔑 关键公式

### 化为定积分

**情形 1：参数方程 $L: x = \varphi(t), y = \psi(t)$**（$t$ 从 $\alpha$ 到 $\beta$，按曲线方向）：
$$\int_L P\, dx + Q\, dy = \int_\alpha^\beta \left[P \varphi' + Q \psi'\right] dt$$

**情形 2：$L: y = f(x)$**（$x$ 从 $a$ 到 $b$）：
$$\int_L P\, dx + Q\, dy = \int_a^b \left[P + Q f'\right] dx$$

**情形 3：空间曲线 $L: x = \varphi(t), y = \psi(t), z = \omega(t)$**：
$$\int_L P\, dx + Q\, dy + R\, dz = \int_\alpha^\beta \left[P \varphi' + Q \psi' + R \omega'\right] dt$$

## ⚠️ 关键特性

- **$dx$ 可正可负**（沿曲线走 $x$ 可能增也可能减）
- **积分值与方向有关**：$\int_{-L} = -\int_L$
- **下限 = 起点对应的参数，上限 = 终点对应的参数**（**不要求 $\alpha < \beta$**！）

## 🔑 与第一型（对弧长）的对比

| 维度 | 第一型 $\int f\, ds$ | 第二型 $\int P\, dx + Q\, dy$ |
|------|----------------------|------------------------------|
| $ds$ vs $dx$ | $ds > 0$ | $dx$ 可负 |
| 方向 | 无关 | 有关 |
| 几何 | 弧长元素 | 坐标投影元素 |
| 物理 | 弧长、线密度 | **功、流量** |

## 💡 做题套路

> 🔑 **四步走**：
> 1. 看曲线形式 → 选参数化方式
> 2. 算 $dx, dy$（用参数表示）
> 3. 代被积函数
> 4. **按曲线方向定上下限**（起点在下，终点在上）

> ⚠️ **折线 = 分段**，每段单独算再求和
