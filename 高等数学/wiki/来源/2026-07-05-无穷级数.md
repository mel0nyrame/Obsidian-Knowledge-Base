# 第 7 章 — 无穷级数

## 📖 章节概要

把"无穷多个数 / 函数相加"的问题系统化。

主要内容：
1. **常数项级数**：判别敛散性、求和
2. **正项级数判别法**：比较 / 比值 / 根式 / 积分
3. **任意项级数**：交错级数（莱布尼茨）、绝对 / 条件收敛
4. **幂级数**：收敛域、收敛半径、和函数
5. **函数展开成幂级数**（泰勒、麦克劳林）
6. **傅里叶级数**

## 🔑 关键判别法速查

| 类型 | 判定方法 |
|------|---------|
| 几何级数 | 看 $\|q\|$：$<1$ 收敛，$\geq 1$ 发散 |
| $p$ 级数 | 看 $p$：$>1$ 收敛，$\leq 1$ 发散 |
| 含阶乘 | 比值法 |
| 含 $n$ 次方 | 根式法 |
| 分式型 | 比较 / 比较极限 |
| 交错 | 莱布尼茨（$u_n$ 单调 $\to 0$）|
| 任意项 | 先绝对 → 再条件 → 再发散 |
| 幂级数 | 阿贝尔 + 比值法求 $R$ + 判端点 |

## 🔑 必背展开式

$$e^x = \sum \frac{x^n}{n!} \quad |x| < +\infty$$
$$\sin x = \sum \frac{(-1)^n}{(2n+1)!} x^{2n+1} \quad |x| < +\infty$$
$$\cos x = \sum \frac{(-1)^n}{(2n)!} x^{2n} \quad |x| < +\infty$$
$$\ln(1+x) = \sum \frac{(-1)^{n-1}}{n} x^n \quad -1 < x \leq 1$$
$$\frac{1}{1-x} = \sum x^n \quad |x| < 1$$
$$\arctan x = \sum \frac{(-1)^n}{2n+1} x^{2n+1} \quad |x| \leq 1$$

## 🎯 做题套路速查

### 判别敛散性
1. 看通项 → 若 $\not\to 0$ → 发散
2. 看类型：
   - 几何 / $p$ 级数 → 直接判
   - 含阶乘 → 比值法
   - 含 $n$ 次方 → 根式法
   - 其他 → 比较法
3. 任意项 → 先看 $\sum|u_n|$，再莱布尼茨

### 求幂级数收敛域
1. 算 $\rho = \lim |a_{n+1}/a_n|$
2. $R = 1/\rho$（$\rho = 0$ 时 $R = +\infty$）
3. 单独判端点 $x = \pm R$
4. 写出收敛域

### 函数展开成幂级数
1. **优先用间接法**（套已知展开式）
2. 通过求导、积分、换元得到目标
3. 写出收敛域
