# 第 3 章 — 多元函数微分学

## 📖 章节概要

把一元函数微分学推广到多元函数。本章核心：
- **偏导数 / 全微分**：把"导数"概念推广到多变量
- **复合函数求导**：链式法则的多元版本
- **隐函数求导**：$F(x, y, z) = 0$ 求导
- **多元极值**：无条件和条件极值

## 🔑 关键公式

### 偏导数与全微分
- 偏导：$\dfrac{\partial f}{\partial x} = \lim_{\Delta x \to 0} \dfrac{f(x + \Delta x, y) - f(x, y)}{\Delta x}$
- 全微分：$df = \dfrac{\partial f}{\partial x} dx + \dfrac{\partial f}{\partial y} dy + \dfrac{\partial f}{\partial z} dz$
- 必要条件：$\dfrac{\partial f}{\partial x} = \dfrac{\partial f}{\partial y} = \dfrac{\partial f}{\partial z} = 0$

### 多元复合求导（链式法则）
- $z = f(u, v), u = \varphi(x, y), v = \psi(x, y)$：
$\dfrac{\partial z}{\partial x} = \dfrac{\partial f}{\partial u} \dfrac{\partial u}{\partial x} + \dfrac{\partial f}{\partial v} \dfrac{\partial v}{\partial x}$

### 隐函数求导
- $F(x, y, z) = 0 \implies \dfrac{\partial z}{\partial x} = -\dfrac{F_x}{F_z}, \dfrac{\partial z}{\partial y} = -\dfrac{F_y}{F_z}$

### 多元极值（充分条件）
- $A = f_{xx}, B = f_{xy}, C = f_{yy}, \Delta = AC - B^2$
- $\Delta > 0, A > 0$ → 极小值
- $\Delta > 0, A < 0$ → 极大值
- $\Delta < 0$ → 非极值
- $\Delta = 0$ → 不确定

## 🎯 做题套路速查

- **求偏导**：固定其他变量，按一元求导
- **复合函数求导**：画出变量关系图（树状）→ 沿链相乘、沿并相加
- **隐函数求导**：用 $F_x, F_y, F_z$ → 套公式
- **判多元极值**：先求驻点 → 再判 $\Delta$ 和 $A$
