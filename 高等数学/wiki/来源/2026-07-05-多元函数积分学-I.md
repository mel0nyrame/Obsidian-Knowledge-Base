# 第 5 章 — 多元函数积分学 (I)

## 📖 章节概要

第一型积分——被积函数"乘以正的"积分元素（$d\sigma, dV, ds, dS$），积分值与方向**无关**。

主要内容：
1. **二重积分**：直角坐标（X 形 / Y 形）+ 极坐标 + 一般变量代换
2. **三重积分**：直角坐标（先二后一 / 先一后二）+ 柱面坐标 + 球面坐标
3. **重积分的应用**：曲面面积、质心、转动惯量、引力
4. **对弧长的曲线积分**（第一型）
5. **对面积的曲面积分**（第一型）

## 🔑 关键公式速查

| 类型 | 积分元素 | 化为 |
|------|---------|------|
| 二重积分 | $d\sigma$ | 二次积分 / 极坐标 |
| 三重积分 | $dV$ | 三次积分 / 柱面 / 球面 |
| 对弧长曲线积分 | $ds = \sqrt{x'^2 + y'^2}\, dt$ | 定积分 |
| 对面积曲面积分 | $dS = \sqrt{1 + z_x^2 + z_y^2}\, d\sigma$ | 二重积分 |

## 🎯 做题套路速查

### 二重积分
- **X 形**：先 $y$ 后 $x$，对每个 $x$，$y$ 从下边界到上边界
- **Y 形**：先 $x$ 后 $y$，对每个 $y$，$x$ 从左边界到右边界
- **极坐标**：区域含圆 / 含 $x^2 + y^2$ → 用极坐标，别忘了乘 $r$

### 三重积分
- **先二后一**：用 $z = $ 常数切片，先算截面二重积分，再对 $z$ 积分
- **柱面坐标**：区域含圆柱、圆锥、$x^2 + y^2$
- **球面坐标**：区域含球、球扇形、$x^2 + y^2 + z^2$

### 第一型曲线/曲面积分
- **$ds = |\text{切向量}|\, dt$**（参数化）
- **$dS = |\text{法向量}|\, d\sigma$**（曲面投影）
