# 第 4 章 — 多元函数微分学的应用

## 📖 章节概要

把多元微分工具用到**几何**和**极值**两大场景。

### 几何应用
- **空间曲线**的切线与法平面
- **空间曲面**的切平面与法线
- **方向导数与梯度**

### 极值应用
- 无条件极值（用第 3 章的充分条件）
- **条件极值**（拉格朗日乘数法）
- 最值问题

## 🔑 关键公式

### 空间曲线（参数方程 $x = x(t), y = y(t), z = z(t)$）
- 切向量 = $(x'(t), y'(t), z'(t))$
- 切线方程：$\dfrac{x - x_0}{x'(t_0)} = \dfrac{y - y_0}{y'(t_0)} = \dfrac{z - z_0}{z'(t_0)}$
- 法平面方程：$x'(t_0)(x - x_0) + y'(t_0)(y - y_0) + z'(t_0)(z - z_0) = 0$

### 空间曲面（一般方程 $F(x,y,z) = 0$）
- 法向量 = 梯度 $(F_x, F_y, F_z)$
- 切平面：$F_x(x - x_0) + F_y(y - y_0) + F_z(z - z_0) = 0$
- 法线：$\dfrac{x - x_0}{F_x} = \dfrac{y - y_0}{F_y} = \dfrac{z - z_0}{F_z}$

### 方向导数
- $\dfrac{\partial f}{\partial \vec{e}_l} = f_x \cos\alpha + f_y \cos\beta + f_z \cos\gamma$
- 其中 $(\cos\alpha, \cos\beta, \cos\gamma)$ 是单位方向向量 $\vec{e}_l$ 的方向余弦

### 条件极值（拉格朗日乘数法）
- 构造 $L = f + \lambda \varphi$
- 解 $\nabla L = 0$ + $\varphi = 0$

## 🎯 做题套路速查

- **曲线切线**：先求切向量（参数式 = 导数，一般式 = 两法向量叉积）
- **曲面法向量**：$\nabla F$（梯度）
- **方向导数**：先单位化方向向量 → 再套公式
- **条件极值**：构造 $L = f + \lambda \varphi$ → 解三元方程组
