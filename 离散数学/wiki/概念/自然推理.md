# 自然推理

## 定义

基于**永真蕴涵式**或**推理规则**进行的命题公式推理，构造一步步从前提到结论的证明过程。

## 12 条永真蕴涵式

| 名称 | 公式 |
|------|------|
| 附加律 | $A \Rightarrow (A \lor B)$ |
| 合取引入 | $(A \land B) \Rightarrow A$ |
| **假言推理（MP）** | $(A \to B) \land A \Rightarrow B$ |
| 假言易位 | $(A \to B) \Rightarrow (\neg B \to \neg A)$ |
| 拒取式 | $(A \to B) \land \neg B \Rightarrow \neg A$ |
| 析取三段论 | $(A \lor B) \land \neg A \Rightarrow B$ |
| 条件三段论 | $(A \to B) \land (B \to C) \Rightarrow (A \to C)$ |
| 双条件三段论 | $(A \leftrightarrow B) \land (B \leftrightarrow C) \Rightarrow (A \leftrightarrow C)$ |
| 合取构造二难 | $(A \to B) \land (C \to D) \land (A \lor C) \Rightarrow (B \lor D)$ |
| 析取构造二难 | $(A \to B) \land (\neg A \to B) \Rightarrow B$ |
| 等值推论 | $(A \leftrightarrow B) \Rightarrow (A \to B)$ |
| 前后件假言推理 | $(A \to B) \land (C \to D) \land (A \lor \neg D) \Rightarrow (B \lor \neg C)$ |

## 3 条推理规则

1. **前提引入**：可随时引入前提
2. **结论引入**：可随时引入已推出的结论
3. **置换规则**：可用等值公式替换子公式

## 证明方法

### 直接证明

从前提出发，用推理规则逐步推到结论。

### 归谬法（反证法）

1. 列出前提
2. 假设 $\neg B$
3. 推出矛盾
4. 得 $B$ 成立

### 归结推理

归结原理：$A \lor B, \neg B \lor C \Rightarrow A \lor C$

- 直接归结：从前提出发反复归结得到结论
- 间接归结：证明 $A_1 \land A_2 \land \ldots \land A_n \land \neg B$ 是矛盾式
