## 对比主题

线性代数提供了两条解线性方程组 $Ax = b$ 的路线：第一章的**克莱默法则**（基于行列式）和第三章的**高斯消元法**（基于矩阵初等变换和秩）。两者解决同一问题，但思想、适用条件和计算效率截然不同。

## 核心对比

| 维度 | 克莱默法则 | 高斯消元法（秩判定法） |
|------|-----------|----------------------|
| **章节** | 第一章 1.5 | 第三章 3.1 / 3.4 |
| **理论基础** | 行列式 | 矩阵初等变换 + 秩 |
| **适用条件** | **仅适用于方阵**（方程数 = 未知数），且 $D \neq 0$ | **适用于任意** $m \times n$ 方程组 |
| **解的判定** | $D \neq 0$ → 唯一解 | $R(A) \neq R(\widetilde{A})$ → 无解；$R(A)=R(\widetilde{A})=n$ → 唯一解；$R(A)=R(\widetilde{A})<n$ → 无穷多解 |
| **唯一解公式** | $x_i = D_i / D$ | 化简行最简形后直接读出 |
| **无穷多解处理** | **无法处理**（只适用于 $n$ 元 $n$ 式且 $D \neq 0$） | 自然地给出通解（含自由未知量） |
| **无解检测** | **无法直接检测**（无解时 $D=0$ 但不一定无解） | 通过 $R(A) \neq R(\widetilde{A})$ 精确判断 |
| **计算复杂度** | $O(n!)$ 行列式计算（大型矩阵不可行） | $O(n^3)$（实用可行） |

## 具体示例

### 方阵唯一解：两者都能处理

方程组 $x_1 + 2x_2 = 5,\ 3x_1 + 4x_2 = 11$：

- **克莱默**：$D = -2 \neq 0$，$D_1 = -2$，$D_2 = -4$，$x_1 = 1, x_2 = 2$ ✓
- **高斯消元**：$\widetilde{A} \rightarrow$ 行最简形，$R(A)=R(\widetilde{A})=2=n$，唯一解 $x_1=1, x_2=2$ ✓

### 非方阵 / 无穷多解：克莱默法则失效

方程组 $x_1 + 2x_2 - x_3 = 0,\ 2x_1 + 4x_2 - 2x_3 = 0$（$m=2, n=3$）：

- **克莱默**：❌ 根本不能使用（不是方阵）
- **高斯消元**：$R(A)=1 < n=3$，有非零解，给出通解 ✓

### 无解情况：克莱默法则误判

方程组 $x_1 + x_2 = 1,\ x_1 + x_2 = 2$：

- **克莱默**：$D = 0$，但只知道「不能保证唯一解」（可能无解也可能无穷多解）⚠️
- **高斯消元**：$R(A)=1$，$R(\widetilde{A})=2$，$R(A) \neq R(\widetilde{A})$，精确判定**无解** ✓

## 结论

| 场景 | 推荐方法 |
|------|---------|
| 小规模 $n \times n$ 方阵，$D \neq 0$ | 克莱默法则（公式优雅） |
| 任意规模方程组 | **高斯消元法**（通用、可计算） |
| 需要判断解的存在性 | 秩判定法（高斯消元路线） |
| 实际数值计算 | 高斯消元法（$O(n^3)$ 远优于 $O(n!)$） |

> 克莱默法则在理论上仍重要（如证明线性系统的性质），但实际求解时高斯消元法是**通用标准方法**。
