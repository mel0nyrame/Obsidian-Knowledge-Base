## 概述

第四章从矩阵代数过渡到向量空间理论，是线性代数从"计算"走向"结构"的关键转折。核心议题：一组向量中哪些是"独立"的？哪些是"多余"的？如何找到最精简的代表团（极大无关组）？最后引入内积概念，将几何直觉（长度、角度、垂直）推广到 $n$ 维。

## 核心要点

### 1. 向量及其运算（4.1）

- **$n$ 维向量**：$n$ 个有序数组成的数组，默认指列向量
- **线性运算**：加法（对应坐标相加）和数乘（每坐标乘 $k$），8 条性质与矩阵一致

### 2. 线性表示与线性相关性（4.2）

#### 线性表示

- **线性组合**：$k_1 a_1 + k_2 a_2 + \cdots + k_s a_s$
- **线性表示**：$a$ 可由 $a_1, \dots, a_s$ 线性表示 $\iff$ $R(A) = R(A, a)$（秩不增加 = 没有新信息）
- **等价向量组**：互相线性表示 → $R(A) = R(B) = R(A, B)$

#### 线性相关性（7 条定理）

- **定义**：存在不全为零的 $k_i$ 使 $k_1 a_1 + \cdots + k_s a_s = 0$ → 线性相关；否则线性无关
- **核心判定**：向量组线性相关 $\iff$ $R(A) < m$（$m$ 为向量个数）；线性无关 $\iff$ $R(A) = m$
- **方阵特例**：$m=n$ 时，$|A|=0$ → 相关；$|A| \neq 0$ → 无关
- **维数 < 个数必相关**：$m$ 个 $n$ 维向量，$n<m$ → 必相关
- **部分与整体**：部分相关 → 整体相关；整体无关 → 部分无关
- **定理 5**：无关组再添一个就相关 → 新来的可被原组唯一表示（极大无关组的理论基础）

### 3. 向量组的秩和极大无关组（4.3）

- **极大无关组**：组内线性无关 + 能表示全家 → "最精简的代表团"
- **向量组的秩** = 极大无关组中向量个数 = 矩阵 $A$（向量按列排）的秩
- **初等变换法求极大无关组**（5 步标准流程）：
  1. 向量按列排成矩阵 $A$
  2. 初等行变换 → 行阶梯形
  3. 主元所在列 → 对应原始向量即极大无关组
  4. 进一步化行最简形
  5. 读出其余向量用极大无关组的线性表示

### 4. 内积与正交（4.4）

- **内积**：$[a, b] = \sum a_i b_i$，满足对称/数乘/加法分配/正定/柯西不等式
- **长度**：$\|a\| = \sqrt{[a, a]}$，单位向量 $\|a\|=1$
- **正交**：$[a, b] = 0$，正交向量组 → 必线性无关
- **正交矩阵**：$A^T A = E$，$|A| = \pm 1$，列（行）构成标准正交向量组
- **正交变换** $y = Px$（$P$ 正交）：保长保角 = 空间的旋转/反射
