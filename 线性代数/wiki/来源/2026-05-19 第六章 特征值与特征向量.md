## 概述

第六章引入方阵的"基因"——特征值与特征向量。它们是方阵最核心的内在属性，决定了矩阵在相似变换下的本质行为。本章从求法出发，到相似对角化，再到实对称矩阵的正交对角化，构成了线性代数最实用的分析工具链。

## 核心要点

### 1. 特征值与特征向量（6.1）

- **定义**：$A\alpha = \lambda\alpha$（$A$ 作用仅拉伸不转向）
- **特征方程**：$|A - \lambda E| = 0$，解出特征值
- **特征多项式**：$\lambda$ 的 $n$ 次多项式
- **核心性质**：
  - $\sum \lambda_i = \operatorname{tr}(A)$（特征值和 = 迹）
  - $\prod \lambda_i = |A|$（特征值积 = 行列式）
  - 不同特征值 → 特征向量线性无关

#### 四种求特征值方法

| 方法 | 适用场景 |
|------|---------|
| 定义法 | 结构简单的矩阵（对角/零矩阵） |
| 特征方程法（最常用） | 标准方法：$|A-\lambda E|=0$ |
| 基本结论法 | 利用迹和行列式反推 |
| 方阵法 | 特殊结构（三角/秩1矩阵） |

#### 四种求特征向量方法

| 方法 | 说明 |
|------|------|
| 定义法 | 直接凑 $A\alpha = \lambda\alpha$ |
| 基础解系法（最常用） | 解 $(A-\lambda_i E)x = 0$ |
| 基本结论法 | 利用不同特征值向量线性无关/正交 |
| 线性无关法 | 判定整体特征向量组的线性无关性 |

### 2. 相似矩阵与对角化（6.2）

- **相似**：$P^{-1}AP = B$，相似矩阵特征值、行列式、迹、秩全相同
- **四种对角化概念**：可相似对角化 / 可正交对角化 / 正交相似矩阵
- **可对角化的四个条件**：
  - 充要：$n$ 个线性无关的特征向量
  - 充要：每个特征值几何重数 = 代数重数
  - 充分：$n$ 个互异特征值
  - 充分：实对称矩阵 → 必可正交对角化

#### 对角化操作步骤

1. 求特征值
2. 对每个特征值求特征向量（基础解系法）
3. 排成 $P$（特征向量按列），写出 $\Lambda$（特征值在对角线）
4. 验证 $AP = P\Lambda$

#### 反求方阵：$A = P\Lambda P^{-1}$（四种方法：定义法/对角矩阵法/实对称矩阵法/谱分解法）

### 3. 实对称矩阵的正交对角化（6.3）

- **定理**：实对称矩阵的特征值全为实数；不同特征值特征向量必正交；**必可正交对角化**
- **主轴定理**：存在正交矩阵 $Q$ 使 $Q^T A Q = \Lambda$
- **正交对角化步骤**：
  1. 求特征值
  2. 求特征向量
  3. 重特征值 → Schmidt 正交化
  4. 单位化所有向量
  5. 构成 $Q$（正交矩阵），写出 $\Lambda$
  6. 验证 $Q^T A Q = \Lambda$（$Q^{-1}=Q^T$，无需求逆！）
