## 概述

第五章从"如何判定解"（Ch3）深入到"解长什么样"——揭示齐次通解与非齐次特解之间的结构关系。核心公式：**非齐次通解 = 一个特解 + 齐次通解**。基础解系作为解空间的极大无关组，是连接 Ch3 秩判定和 Ch4 向量理论的桥梁。

## 核心要点

### 1. 齐次方程组 $Ax = 0$ 解的结构（5.1）

- **解向量**：满足 $Ax = 0$ 的 $x$，加法和数乘封闭 → 构成**解空间**（$\mathbb{R}^n$ 的子空间）
- **基础解系**：解空间的一个极大无关组，含 $s = n - r = n - R(A)$ 个线性无关的解向量
- **通解**：$x = k_1 \xi_1 + k_2 \xi_2 + \cdots + k_{n-r} \xi_{n-r}$（基础解系的所有线性组合）

#### 求基础解系（标准 5 步）

1. 系数矩阵 $A$ → 初等行变换 → 行最简形
2. 确定 $r$（非零行数）、$n$（未知数个数）、$s = n - r$（基础解系含解向量数）
3. 还原同解方程组
4. 自由未知量逐一取单位向量 $(1,0,\dots,0)^T, (0,1,\dots,0)^T, \dots$，解出主变量
5. 写出基础解系和通解

### 2. 非齐次方程组 $Ax = b$ 解的结构（5.1）

- **性质 1**：两个特解之差 → 齐次解
- **性质 2**：特解 + 齐次解 → 仍是非齐次解
- **通解公式**：$x = \eta^* + k_1 \xi_1 + \cdots + k_{n-r} \xi_{n-r}$
- **求通解流程**：先化行最简形 → 令自由未知量 = 0 求特解 → 常数项改为 0 求齐次基础解系 → 合并

### 3. 非齐次与齐次的关系（5.2）

| 非齐次情况 | 对应齐次情况 | 反之成立？ |
|-----------|-------------|-----------|
| $Ax=b$ 唯一解 | $Ax=0$ 只有零解（$R=n$） | **不成立** |
| $Ax=b$ 无穷多解 | $Ax=0$ 有非零解（$R<n$） | **不成立** |

> 非齐次有解的大前提是 $R(A) = R(\widetilde{A})$。齐次恒有解，但非齐次不一定。
