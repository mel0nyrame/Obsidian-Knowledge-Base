# raw/ — 原始资料层

> ⚠️ **只读层**：此目录下的所有文件由**用户**维护，Claude 只读不写。

## 结构

```
raw/
├── README.md          ← 本文件
├── 第一章行列式.md      ← 第一章课堂笔记
├── 第二章矩阵.md        ← 第二章课堂笔记
└── 素材/               ← 文章/笔记内嵌图片的本地存储
```

## 使用方式

1. 将新资料（课本章节、论文、网页文章等）放入 `raw/`
2. 对 Claude 说"摄取 XXX"来触发 wiki 化处理
3. Claude 会读取 raw 文件，在 `wiki/` 中创建摘要、更新概念和索引

## 图片

若资料含图片，下载到 `raw/素材/` 目录，wiki 页面通过 `![[素材/图片名.png]]` 引用。
